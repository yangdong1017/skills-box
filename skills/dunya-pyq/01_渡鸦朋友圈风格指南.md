## 一、渡鸦朋友圈的核心标准

### 品牌人设画像

| 维度 | 标准 |
|------|------|
| 身份 | 琛哥——渡鸦科技社创始人，流量教练+精神领袖 |
| 口吻 | 草根出身、江湖气、直率豪爽、不装、敢说脏话、有哲学和格局 |
| 和读者关系 | 兄弟/战友，不是老师和学生 |
| 口头禅 | "手握流量，兴风作浪" |
| 核心信念 | 用真心做实事给真东西，帮兄弟们赚钱 |

### 好的渡鸦朋友圈 = 什么感觉？

读完了要觉得：
1. **"这兄弟说话够实在"** —— 有干货有案例，不是空话
2. **"卧槽说到我心坎里了"** —— 痛点抓得准，情绪到位
3. **"这哥们有点东西"** —— 有方法论、有洞察、有格局
4. **"我也想加入渡鸦"** —— 社会证明+品牌背书+CTA引导

## 二、内容方向矩阵

| 方向 | 核心功能 | 占比建议 | 内容类型 |
|------|---------|---------|---------|
| **成果晒单** | 建立信任+激发渴望 | ~40% | 会员成功案例、数据战报、晒单截图 |
| **认知输出** | 展示专业深度 | ~25% | 流量方法论、行业洞察、底层逻辑 |
| **品牌动态** | 强化品牌认知 | ~15% | 产品升级、大会、团队里程碑 |
| **社群转化** | 核心转化目标 | ~20% | 引流到588社群、招募、活动预告 |

**最好的朋友圈同时命中两个方向。** 比如：会员晒单（成果）+ 方法论提炼（认知）。

## 三、文案结构骨架

### 骨架A：痛点共鸣型（最常用）
```
{排比3个痛点场景——用"你是不是也……"}
{情绪爆发句——"卧槽""焦虑到头秃"}
{转折——"但问题不在XX，而在XX"}
{方法论一句话——用渡鸦术语}
{金句收尾}
```

### 骨架B：转折叙事型
```
{具体场景切入——"昨天一个兄弟……"}
{情感叙事——不避讳真实困境}
{转折顿悟——"直到他……"}
{认知升级——用"万流归因诀""流量相对论"等}
{金句/号召收尾}
```

### 骨架C：暴击输出型
```
{先给金句/反常识结论}
{一句话拆核心——"本质上是XX"}
{真实案例/数据佐证——精确数字}
{号召收尾——"后台回复【渡鸦社群】"}
```

### 骨架D：品牌叙事型（英雄之旅）
```
{悬念/哲思开场}
{繁荣巅峰/高光时刻}
{困境描述}
{突破转折}
{翻盘展示}
{情感升华}
{"手握流量，兴风作浪！"}
```

## 四、高质量句式参考（渡鸦版）

1. **极致对比句**："你抄1.0的时候，我们已经迭代到8.0！""因为淋过雨，所以想为后来者撑把伞"

4. **口语插入句**："话不多说""兄弟们感受一下！""就问你够不够详细""根本回不完"
5. **自问自答句**："为什么渡鸦增长这么快？琛哥的答案是——用真心，做实事，给真东西"
6. **方括号标注**：【渡鸦洞察】【渡鸦提醒】【渡鸦警示】【渡鸦敲黑板】
7. **古文/诗词混搭句**："世人畏果我畏因""有道无术，术尚可求；有术无道，止于术"

## 五、语言毒药（出现即清洗）

### AI味词汇
- "值得一提""不禁让人""令人深思""毋庸置疑"
- "请记住""真相是""事实上""值得注意的是"
- emoji堆叠超过2个
- 空洞排比（形式整齐但没有信息增量）

### 敬语体
- "您好""请问""感谢您的关注""欢迎咨询"
- 任何像客服的措辞

### 学术腔
- "综上所述""研究表明""笔者以为""由此可见"
- 任何像论文的措辞

### 广告腔
- "限时优惠""错过再等一年""赶快行动""不容错过"
- "限时""抢购""钜惠""震撼来袭"

### 鸡汤味
- 没有具体场景支撑的正能量
- "相信自己""你值得更好的""加油""一切都会好的"
- 没有实战数据背书的道理

### 套路味
- "家人们""这个太绝了""不允许还有人不知道"
- "保姆级教程""建议收藏""码住"
- 标题党式开头

### 软弱表达
- "或许""可能""也许""尽量""大概""差不多"

## 七、字数与节奏

| 类型 | 字数 | 适用场景 |
|------|------|---------|
| 短文案 | 50-200字 | 日常战报、金句、晒单 |
| 中文案 | 200-400字 | 有感而发、方法论、会员案例 |
| 长文案 | 400-600字 | 深度输出、品牌叙事、大会复盘 |
| 超长 | >600字 | 转公众号长文 |

**节奏原则**：
- 每段不超过4行
- 关键信息单独成行
- 大量使用空行制造呼吸感
- 金句必须单独一行
- 口语↔专业之间自然切换，不突兀

## 九、自检清单

每次写完朋友圈后，对照以下清单检查：

- [ ] 开头是否有"兄弟们"+ 痛点共鸣/场景代入？
- [ ] 是否有渡鸦品牌术语（渡鸦/琛哥/截流/自热/私域/矩阵等）？
- [ ] 语气是否在口语↔专业之间自然切换？
- [ ] 是否至少包含1个军事/武侠/江湖隐喻？
- [ ] 是否至少包含1句金句？
- [ ] 是否有会员案例/数据作为社会证明？（成果晒单类必须有）
- [ ] 结尾是否带CTA或"手握流量，兴风作浪"？
- [ ] 段落是否足够短（每段<4行）？
- [ ] 是否没有使用敬语体/学术腔/广告腔/AI味？
- [ ] 社群转化类文案中价格是否为588？
