# dunya-pyq

渡鸦科技社专属朋友圈文案 Skill。

它用于把渡鸦知识库、赛道 SOP、会员案例、公众号方法论和品牌语气，封装成可在 Claude Code 中直接触发的朋友圈文案引擎。

## 能做什么

- 生成渡鸦品牌风格朋友圈文案
- 诊断并重写已有朋友圈文案
- 快查行业/赛道打法，并生成可直接发布的朋友圈版本
- 根据知识库素材提炼私域引流、社群转化、会员战报、品牌动态类内容

## 典型触发

```text
/dunya-pyq
帮我写一条朋友圈
这条朋友圈怎么样，帮我优化
帮我查一下医美赛道
婚恋/情感复合赛道怎么打
今天有个会员跑通了小红书截流，帮我发朋友圈
```

## 安装

复制整个目录到 Claude Code 的用户 skills 目录：

```text
C:\Users\<你的用户名>\.claude\skills\dunya-pyq
```

或放到项目级 skills 目录中使用。

## 使用方式

在 Claude Code 中直接输入：

```text
帮我写一条朋友圈：今天有个做医美的会员反馈，小红书评论区截流跑通了，3天加了80多个精准咨询。
```

或：

```text
这条朋友圈怎么样，帮我诊断并重写：大家好，我们的社群课程正在限时优惠中……
```

## 发布前注意

这个 Skill 的价值主要来自 bundled knowledge base。推送到公开 GitHub 前，请确认：

- 公众号原文、会员案例、诊断方案是否允许公开发布
- 是否需要移除联系人、微信号、二维码、价格和内部运营数据
- 是否需要改成私有仓库或只发布脱敏版

如果只是个人多设备同步或团队内部使用，建议使用私有仓库。
