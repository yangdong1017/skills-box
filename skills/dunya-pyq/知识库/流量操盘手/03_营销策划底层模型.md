# 第3章 营销策划底层模型：把想法变成销售

## 章开：一场失败的产品发布

一个产品经理花了三个月时间打磨一个新产品，定价999元。上线第一天，他在公司的微信群、自己的朋友圈、几个行业社群里发了同一条文案，告诉大家"我做了一个牛逼的产品，你们买吧"。

结果怎样？一天0个购买。一周0个购买。一个月过去，还是0个。

他很困惑。产品这么好，为什么没人买？他开始找朋友倾诉，朋友听完后说了一句话："你有没有想过，人家根本不知道这个产品能解决什么问题？你又没有展示过用户的真实需求，没有讲过你产品的差异在哪里，也没有证明过你能提供比对手更好的价值。你只是说'这很牛逼'，但为什么牛逼呢？牛逼在哪里呢？"

与此同时，另一个产品经理也推出了一个类似的产品，定价同样是999元。但他花了两周时间做了完整的营销策划。他调研了用户的痛点，采访了十个目标客户，研究了三个竞争对手，最后制定了一个清晰的定位策略和销售路径。

他的上线方式是：先在知乎上发了一篇"为什么你的Excel永远学不好"，获得了10000个赞。然后邀请这些有兴趣的人进微信群，在群里分享了5个免费工作坊，建立了信任。最后才正式推销产品。

结果呢？虽然第一个月也没有大的爆炸，但他已经成交了15个客户，而且得到了4个5星评价。通过口碑转介，第二个月他成交了40个客户。

**这就是有没有营销策划底层思维的差别。**

## 营销的四个层级

营销是一个很宽泛的概念。有的人理解的营销是"写文案"，有的人理解的是"做广告"，有的人理解的是"做内容"。这些都对，但都不全面。

如果你把这些片段式的理解加起来，就会陷入一个常见的陷阱：花了很多时间在"怎样写更好的文案"或"怎样拍更吸引人的视频"，但整个营销策略仍然没有方向。就像一辆车，发动机再好，如果没有方向盘和地图，也开不到目的地。

我把营销分成四个层级，从下到上分别是：底层是定位与洞察，中间是策略制定和执行优化，顶层是数据与决策。这四个层级是递进的。如果底层（定位）不清楚，上面的所有工作都是白费。

### 第一层：定位与洞察（地基）

这是一切营销的基础。你必须清楚地回答这四个问题，这四个问题构成了营销的底层基石：

```
                  ┌──────────────────────────┐
                  │    营销四问体系          │
                  └──────────────────────────┘
                            │
           ┌────────────────┼────────────────┐
           │                │                │
        ┌──▼──┐        ┌────▼────┐     ┌────▼────┐
        │产品 │        │客户群体 │     │核心痛点 │
        │属性 │        │定位     │     │洞察     │
        └─────┘        └─────────┘     └─────────┘
           │                │                │
           │            ┌────▼────┐         │
           │            │竞争优势 │         │
           │            │差异化   │         │
           │            └─────────┘         │
           │                │                │
           └────────────────┼────────────────┘
                            │
                  ┌─────────▼─────────┐
                  │清晰的营销定位     │
                  │（指导所有行动）   │
                  └───────────────────┘
```

1. **我的产品是什么？** 不是说名字，而是说本质和能为用户带来什么。比如，不要说"我卖的是一个Excel课程"，这太肤浅了。要说"我帮助上班族用3个月的碎片时间掌握数据分析能力，从而提升工作效率30%、加薪20%"。注意，我没有说"Excel课程很专业"这样的自我赞美，而是说"能带来什么具体的结果"。用户买的不是产品，而是结果。

2. **我的客户是谁？** 具体的人群描述，不是"想学Excel的人"这样的泛泛之谈。而是"28-38岁、在公司工作3-8年、月薪8K-20K、想晋升但觉得技能不足的一二线城市职场女性"。越具体越好。具体到什么程度？具体到你能凭着这个描述，在街上一眼认出这个人。

3. **客户的核心痛点是什么？** 不是所有的痛点，而是那个最紧迫、最能驱动购买决策的那个。比如，很多人想学Excel，但最驱动他们购买的，可能不是"我想学好Excel"（这太温和了），而是"我的同事因为会Excel升职了，我眼看着被拉开了距离，再不学就要被淘汰了"（这是焦虑和恐惧）。抓住这个痛点，你的文案就会更有力度。

4. **我凭什么比竞争对手更好？** 这个叫做"竞争优势"。你的优势可能是价格、质量、服务、专业性、背景故事等。但关键是，必须选一个能清楚传达给用户的优势，而不是你觉得自己很全能（全能往往意味着没有特别）。比如，不要说"我的课程既便宜、质量又好、服务又周到"。这样说，用户会觉得这是"营销话术"，反而降低信任。更聪明的做法是选一个主要优势并深化它。比如"我的课程是业界唯一保证学完3个月内不用学就能在工作中直接应用的"。

只有回答清楚这四个问题，你才有了真正的"定位"。定位不是一个虚的概念或漂亮的标语，而是指导所有营销行动的指南针。接下来你写的每一句文案、发的每一条视频，都应该围绕这个定位。如果某个内容偏离了定位，就要删掉，不管它有多漂亮。

下表总结了STP模型（细分→目标→定位）的三个核心要素，帮助你系统地完成定位工作：

| 要素 | 定义 | 关键问题 | 常用工具 |
|------|------|--------|--------|
| **细分（Segment）** | 识别市场中不同的客户群体，找出具有共同需求的细分市场 | 市场有哪些不同的人群？他们的需求有什么差异？我应该聚焦哪个细分？ | 5118关键词分析、竞品分析、用户调研 |
| **目标（Target）** | 在众多细分中，选择最有吸引力、最符合自己优势的目标市场 | 我的目标客户是谁？为什么选他们而不是其他人？这个市场有多大？ | 市场规模评估、竞争强度分析、盈利潜力评估 |
| **定位（Positioning）** | 在目标客户心中建立独特的、有差异的品牌形象和价值认知 | 我在目标客户心中想占据什么位置？我的竞争优势是什么？我怎样传达这个优势？ | 品牌故事塑造、价值主张设计、传播渠道选择 |

### 第二层：策略制定（框架）

有了定位，下一步是制定策略。策略回答的是"我怎样用最高效的方式传达这个定位"。整个营销策略的制定，需要经历以下流程：

```
┌──────────────────┐
│  市场分析结果    │
│  用户画像清晰    │
└────────┬─────────┘
         │
         ▼
┌──────────────────────────────────┐
│  确定营销目标                    │
│  （品牌认知/获客/复购/转介绍）    │
└────────┬─────────────────────────┘
         │
         ▼
┌──────────────────────────────────┐
│  选择营销渠道                    │
│  （知乎/微信/抖音/小红书等）      │
└────────┬─────────────────────────┘
         │
         ▼
┌──────────────────────────────────┐
│  规划内容策略                    │
│  （干货/故事/转化内容比例）       │
└────────┬─────────────────────────┘
         │
         ▼
┌──────────────────────────────────┐
│  制定转化路径                    │
│  （多少个环节/每个环节的目标）    │
└────────┬─────────────────────────┘
         │
         ▼
┌──────────────────────────────────┐
│  评估获客成本目标                │
│  （CAC/LTV/毛利润测算）          │
└────────┬─────────────────────────┘
         │
         ▼
┌──────────────────┐
│  执行营销方案    │
│  （内容/渠道）    │
└──────────────────┘
```

很多人会在这一层犯错。他们会说"我的策略就是发很多高质量内容"。听起来不错，但这不是策略。策略应该更具体，具体到"什么时间、什么渠道、什么内容、怎样转化"。

好的策略通常包括四个部分，下表总结了4P营销组合的关键决策维度：

| 营销要素 | 关键决策 | 考虑因素 | 目标用户影响 |
|--------|--------|--------|-----------|
| **产品（Product）** | 产品形式、品质、特性、包装、品牌 | 目标用户的需求、竞品对标、成本控制 | 决定用户的第一印象和价值认知 |
| **价格（Price）** | 定价策略、折扣方式、支付条件、促销政策 | 客户购买力、竞品价格、目标利润 | 影响购买决策和客户获取成本 |
| **渠道（Place）** | 销售渠道、分销体系、触达方式、合作伙伴 | 目标用户在哪里、竞争现状、效率成本 | 决定用户能否看到和购买产品 |
| **推广（Promotion）** | 广告、文案、内容、公关、促销活动、口碑 | 用户习惯、信息偏好、心理驱动、阶段特征 | 影响品牌认知、需求激发、购买欲望 |

**1. 渠道选择策略**

你选择在哪些渠道上发力？为什么不是所有渠道都投？最重要的原因是：**你的目标客户在哪里，你就该在哪里。**

举个例子。如果你是职场人士教Excel，知乎和微信是最好的渠道。为什么？因为职场人在知乎上搜索"怎样学Excel"，在微信上接收朋友的推荐。但抖音可能就不是首选。为什么？因为抖音用户虽然多（10亿），但他们的来抖音的心态是"娱乐"而不是"学习技能"。即便你的Excel视频火了，转化率也会很低。

所以，正确的思路不是"哪个平台用户多就去哪个"，而是"哪个平台我的目标用户会看"。

**2. 内容策略**

在你选择的渠道上，你要发什么内容？是干货分享、案例故事、知识讲解，还是日常分享？不同的内容会吸引不同的人，也会产生不同的信任度和转化率。

比如，你在知乎发干货（"Excel 10大高效技巧"），会吸引那些已经意识到"我需要学Excel"的人。但如果你发故事（"我从Excel白痴到公司数据分析师的经历"），会吸引那些还在"想要改变"但还没行动的人。哪个更重要？取决于你的用户阶段。初期应该发故事来吸引有痛点的人。有了一定规模后，可以发干货来进一步建立专业性。

**3. 转化策略**

从看到你的内容，到最终购买，中间应该经过几个转化环节？比如：
- 第一环节：在知乎看到你的文章，产生兴趣
- 第二环节：在文章底部留下微信号或点击链接
- 第三环节：进入微信群，接收一周的免费干货
- 第四环节：参加你的直播讲座
- 第五环节：最终购买

每个环节都有目的和工具。第一环节的目的是"吸引有需求的人"，工具是"高质量的内容"。第二环节的目的是"建立一对一的联系"，工具是"微信号和微信群"。等等。

**4. 客户获取成本目标**

基于漏斗反推法，你的获客成本应该控制在多少以内，才能保证利润？比如，如果你的客单价1980，毛利60%（1188），你希望获客成本不超过400块，这样每个客户的净利润就是788块。这个数字会指导你在营销上该花多少钱、该选择什么渠道（低成本还是高转化）。

### 第三层：执行与优化（行动）

有了定位和策略，下一步是实际执行。执行包括：

- **创建内容**：写文章、拍视频、录音频等。这里的关键是"质量和频率的平衡"。不要为了频率而降低质量，也不要为了完美而永远写不出来
- **投放内容**：发到选定的渠道，可能需要付费（比如抖音广告），也可能免费（比如知乎文章）
- **收集反馈**：用户的点赞、评论、分享、购买等数据都是反馈。关键是"系统地收集"，而不是凭印象
- **持续优化**：根据数据调整内容形式（什么样的标题吸引人？）、渠道策略（这个渠道是否值得继续投入？）、文案话术等

这个过程不是线性的，而是循环的。你发一批内容，看反馈，调整，再发，再看反馈。通过不断的小循环，逐步找到最高效的方式。

执行层的一个常见错误是"追求完美"。有的人说"等我写出完美的10篇文章，再一次性发布"。结果等了3个月还没写好，市场已经变了。更好的做法是"先发一个80分的版本，看用户反应，再迭代"。速度往往比完美度更重要。

### 第四层：数据与决策（大局）

最后一层是把所有的执行数据综合起来，做出更大的战略决策。

比如，你发现在知乎上的转化率是5%、成本很低，在小红书上是2%、成本很高。那你应该把更多的资源投向知乎，甚至考虑停止在小红书的投放。

或者，你发现付费流量（广告）的获客成本是2000块，但口碑转介的获客成本是0（因为老客户带来新客户不需要额外的营销成本）。那你应该把资源从"获取新客"转移到"促进老客复购和转介"上。

再比如，你发现女性客户的购买力和复购率都比男性高，那你应该调整定位，从"所有职场人"改为"职场女性"，并在这个细分市场上深耕。

这四层是递进的关系。很多人花了很多精力在第三层（执行）和第四层（数据），但第一层和第二层没有想清楚。结果是，他们会优化得很辛苦（把一个不合适的渠道优化到不能再优化），但整体的效果还是有限。

反过来，如果你的第一层（定位）清晰，第二层（策略）合理，再花中等的努力在第三层执行，效果往往会很好。这就是"战略的力量"。

下图展示了营销四个层级的递进关系和每一层的核心工作：

```
                    ┌─────────────────────────┐
                    │   第四层：数据与决策    │
                    │  • 综合分析所有数据     │
                    │  • 做出战略调整         │
                    │  • 优化资源分配         │
                    └────────────┬────────────┘
                                 │ 指导
                    ┌────────────▼────────────┐
                    │  第三层：执行与优化     │
                    │  • 创建内容             │
                    │  • 投放到渠道           │
                    │  • 收集反馈             │
                    │  • 持续迭代             │
                    └────────────┬────────────┘
                                 │ 依据
                    ┌────────────▼────────────┐
                    │  第二层：策略制定       │
                    │  • 渠道选择             │
                    │  • 内容规划             │
                    │  • 转化设计             │
                    │  • 成本目标             │
                    └────────────┬────────────┘
                                 │ 基于
                    ┌────────────▼────────────┐
                    │  第一层：定位与洞察     │
                    │  • 产品定义             │
                    │  • 客户定位             │
                    │  • 痛点理解             │
                    │  • 竞争优势             │
                    │  (营销的地基)           │
                    └─────────────────────────┘

        ⚠ 关键：底层不清，上层白费 ⚠
```

## 竞品分析：了解对手赢的方式

现在讲一个很多营销新手忽视的工作：竞品分析。

很多人听到"竞品分析"，就想到了抄袭。"分析对手，不就是想学他们怎么做的吗？"

这理解是错的。竞品分析的目的，不是抄袭，而是理解。理解对手为什么这样做，他们的优势在哪里，他们的劣势在哪里，从而找到自己的差异化机会。

好的竞品分析，包括三个部分：

### 1. 广义竞品的定义

首先，你需要理解一个重要的概念：**竞品不一定是形态相同的产品。**

比如，一个职场人想学Excel。他的选择包括：报名一个Excel课程（499元）、买一本Excel教材（89元）、看YouTube上的免费教程、问同事帮助、找个Excel高手一对一指导等。

这些都是"竞品"吗？是的。因为它们都在解决同一个需求：学Excel。

理解这一点很重要。因为很多时候，你的真正竞争对手，不是同行的课程，而是"用户选择不付费学习"或"用户选择付费找一对一指导"。所以，你不仅要对标同行的课程，还要理解用户为什么不选择免费方案，为什么不选择其他付费方案。

### 2. 竞品的商业模式分析

如果你有竞品，你要分析的不仅是他们的产品，还有他们的商业模式。

具体来说，包括：

**价格策略**：他们的定价是多少？为什么这么定价？是走"高端、专业、贵"的路线，还是"平民、便宜、高转化"的路线？

**用户获取方式**：他们的客户来自哪里？是广告投放、口碑转介、内容营销、销售电销，还是别的？这说明了他们在营销上的侧重点。

**转化策略**：他们是直接销售（看产品页立刻下单），还是有多个转化环节？

**复购模式**：他们怎样让用户重复购买？是产品本身有高复购率，还是通过社群维系？

**渠道布局**：他们在哪些平台有存在？是只在一个平台，还是多渠道布局？

通过分析这些，你会看到竞品的整个商业版图。这给你很多启发：他们做对了什么，你能否学习？他们遗漏了什么，你能否弥补？

### 3. 竞品数据的获取

该怎样获取竞品的数据呢？有几个工具和方法：

**抖音数据工具**（如馋妈妈）：查看竞品在抖音的销售数据、直播数据、粉丝互动等。这能告诉你竞品在抖音上有多成功。

**搜狗搜索**：搜索竞品的微信公众号，看他们的粉丝数、内容更新频率、内容质量等。

**电商平台**：在淘宝、京东等平台搜索竞品，看销量、评价、定价等。

**知乎**：搜索相关的关键词，看竞品或竞品的创始人有没有活跃的内容。

**直接交互**：报名竞品的课程或社群，体验他们的销售流程、课程质量、客户服务等。这往往能给你最直接的感受。

通过这些渠道，你能拼出一个比较完整的竞品画像。

## 用户洞察：找到痛点而不是需求

很多营销人说"我要做用户调研"，结果就是发一个问卷："你想学什么？" 然后看到很多人回答"我想学Excel"。

这样调研有什么用？没什么用。因为你已经知道人们想学Excel了。真正的洞察，不在于"需求是什么"，而在于"为什么现在买不起、为什么以前没买、买之后能改变什么"。

好的用户洞察，包括两部分：

### 问题的深度

一个人说"我想学Excel"，这叫做"需求的表面"。更深一层，也许他真正的痛点是"我工作效率低，同事比我快，我怕被边缘化"。再深一层，也许他的真正焦虑是"我想晋升，但没有差异化竞争力"。

不同的深度，会导致不同的营销策略。如果你理解到第一层，你的文案会是"学习Excel工具"。如果你理解到第二层，你的文案会是"用Excel提高工作效率，不被同事比下去"。如果你理解到第三层，你的文案会是"Excel高手是职场竞争的门槛，掌握它，你才能突出重围"。

显然，第三层的文案更能触发购买欲望。

### 问题的来源

用户的痛点来自哪里？是他对自己的不满意吗？是竞争压力吗？是生活变化吗？

比如，一个学生学Excel和一个职场人学Excel，痛点可能是不同的。学生也许是为了"应付考试"，职场人也许是为了"升职加薪"。你的营销策略应该针对不同的痛点来设计。

## 定位与差异化

现在我们从洞察落地到实际的定位工作。

定位不是一个虚的概念，也不是"漂亮的广告语"。一个好的定位，应该：
1. 能让你在竞争中脱颖而出
2. 能指导你所有的营销行动
3. 能被目标客户清晰地感受到

### 定位的三个要素

**1. 目标人群**

你的产品是给谁的？不要说"所有想学Excel的人"，这太宽泛了。要说"28-38岁的一二线城市职场女性，在公司工作3-8年，月薪8K-20K，想晋升但觉得自己竞争力不足"。

为什么要这么具体？因为具体的定位能帮你：
- 更清晰地选择渠道（你知道这些人在哪里活跃）
- 更精准地写文案（你知道她们的语言风格和痛点）
- 更有效地成交（你的推销话术会更有针对性）

**2. 核心价值主张**

你给这个人群的核心价值是什么？不要说"高质量的课程"或"业界最强的师资"，这些都是虚的。要说具体的"结果"。

比如，不要说"我教Excel"，要说"在3个月内把你从'用不好Excel的普通员工'变成'公司数据分析的小能手，能独立完成报表分析，从而在升职时更有竞争力，有机会加薪25%'。"

注意，这个价值主张包含了：时间周期（3个月）、改变（从XX到XX）、具体的能力（独立完成报表分析）、最终的结果（加薪25%）。用户读到这个，会想象自己学完后的样子。这就是有说服力的价值主张。

**3. 竞争优势**

为什么他们应该选你而不是别人？你的竞争优势是什么？可能的方向包括：
- **经历优势**："我曾经也是Excel白痴，现在年薪翻倍，我深知你的困扰"
- **方法优势**："我的教学方式是业界唯一的3个月快速法，其他课程都要半年"
- **结果优势**："我帮助过500名学生升职，其中80%在学完后3个月内加薪"
- **服务优势**："除了课程，我还提供一年的职业咨询，帮你规划升职路线"
- **价格优势**："我的课程只要1980，比同行便宜50%，但效果相同"

选择一个最强的、最能表达的优势，把它作为你的主要卖点。不要试图在"经历、方法、结果、服务、价格"上都超过别人，这是不现实的。专注于一个，把它做到极致。

### 差异化的五个方向

在制定定位时，你可以从这五个方向找到差异化：

**1. 人群差异**

我不是做"所有想学Excel的人"，我只做"小公司的财务人员"或"销售经理"。更细分的人群，会有更强的认同感，因为你的例子、故事都是针对他们的。

**2. 价值差异**

别人说"学好Excel"（太模糊），我说"用Excel把财务报表从4小时的手工操作减到1小时的自动化，还能避免99%的计算错误"。更具体、更可衡量的价值，用户能更清楚地评估"这对我值不值得"。

**3. 方式差异**

别人是录播网络课程（学员很孤独），我是直播小班（有互动有答疑）。或者别人是理论讲授，我是实战项目制（学完就能用）。不同的教学方式，会吸引不同偏好的学生，也能形成差异。

**4. 服务差异**

别人的课程就是课程（学完就完了），我还提供一年的"职业发展社群"和"升职咨询"。额外的服务，能大幅提升用户的满意度和复购率，也是竞争优势。

**5. 品牌差异**

别人是"刚毕业的新讲师"，我是"在微软做了5年数据分析、后来创办了3家公司、帮助过500名学生升职的资深人士"。背书和故事，能形成品牌信任度的差异。

### 选择和验证定位

制定完定位后，别急着全面投放。先小范围验证一下。比如：
- 写10条广告文案，基于这个定位
- 看你的目标用户（比如在知乎上找到的人）会不会点击、会不会评论
- 看他们的反应是否符合你的期待

如果反应冷淡，说明定位可能有问题。此时调整定位，再试。不要投入大量资源来推广一个还没验证过的定位。

## 内容规划与故事框架

有了定位，下一步是规划内容。内容是传达定位、驱动转化的媒介。

### 内容的三种角色

**1. 教育内容**

目的是证明你有专业性。比如，你写一篇"Excel函数的7个高级用法"，展示你确实懂Excel。

**2. 故事内容**

目的是产生情感共鸣。比如，你讲一个"我曾经不会Excel，错过了升职，痛定思痛后学会了，现在年薪翻倍"的故事。这个故事会让有相同经历的人产生认同感。

**3. 转化内容**

目的是直接驱动购买。比如，你在某个地方（知乎、微信群等）明确说出你的产品、价格、保障等，然后给出购买链接。

一个完整的营销内容规划，应该这三种内容都有，而且有合理的比例。通常来说，可以是4:3:1——40%教育内容，30%故事内容，10%转化内容。

不要所有内容都是转化内容，那样会让人反感。也不要都是教育内容，那样没人会买。

### 故事框架

在讲故事时，一个好的框架能让故事更有说服力。一个有效的故事框架应该包含三个关键部分：

**第一部分：问题**

讲一个你之前遇到的问题。这个问题最好能引起目标用户的共鸣。比如，"我曾经工作效率很低，经常加班到晚上10点，周末还要补工作"。这个部分的目的是让听众产生代入感——他看到你过去的问题，想到了自己。

**第二部分：行动**

讲你为了解决这个问题做了什么。比如，"我开始学习Excel的高级函数和数据透视表，用了3个月的时间"。这个部分建立了一个"解决路径"，让听众知道可以通过什么方式来改变现状。

**第三部分：结果**

讲最终的结果是什么。比如，"现在我用Excel自动化了80%的重复工作，每天下午5点就能下班，年薪也从25万升到了35万"。这个部分是故事的高潮——展示了问题被解决后的美好状态。

这个框架之所以有效，是因为它有一个完整的因果链：有问题 → 解决问题 → 得到结果。听众会自然地想象，"如果我也这样做，我也能得到这样的结果"。这就是故事的说服力所在。

## 本章小结

营销策划的底层，就是搞清楚四个问题：

**第一：我是谁？** 你的产品本质、目标人群、核心痛点、竞争优势各是什么？这是定位。这层做不清楚，后续所有努力都是在错误的方向上。

**第二：我在哪里？** 我的目标客户在哪些平台、社群上活跃？我应该在哪些渠道上发力，用什么样的内容？这是策略。好的策略能让有限的资源发挥最大的效果。

**第三：我要说什么？** 我用什么内容、什么故事、什么话术来传达我的价值？我如何从"有了曝光"变成"有了成交"？这是执行。

**第四：效果怎样？** 通过什么数据来衡量营销效果，哪些工作有效哪些无效，下一步怎么优化？这是优化和迭代。

把这四个问题都回答清楚了，你的营销就不会"碰运气"，而是"有规律"了。更重要的是，你会对自己的营销有足够的"控制感"。而不是被动地等待流量、等待转化，而是主动地设计流量、设计转化路径。这个控制感，往往就是区分"业余创业者"和"专业创业者"的根本。

## 操盘手练习

**练习1：定位清晰化**

为你的产品或服务做一个完整的定位：

1. 写一个30字以内的目标人群描述（不要说"所有人"，要具体）
2. 写一个50字以内的核心价值主张（解决什么问题，产生什么结果）
3. 列出你的三个主要竞争优势
4. 在这三个优势中，选择一个最容易传达、最能引发购买欲望的，作为你的主打定位

完成这个练习后，你就有了一个清晰的、可以指导营销的定位。

**练习2：竞品分析**

选择你的一个主要竞争对手，进行深度分析：

1. 他们的定位是什么？（目标人群、价值主张、竞争优势）
2. 他们的产品定价是多少？
3. 他们在哪些渠道上获客？（抖音、知乎、直播等）
4. 他们的销售流程是什么？（从看到内容，到成交，经过了哪些环节）
5. 相比他们，你有什么优势和劣势？
6. 有没有被他们忽视的市场空隙，你可以填补？

通过这个练习，你会更清楚自己的竞争环境，以及该从哪个方向做差异化。

**练习3：用户洞察访谈**

约5-10个目标客户（最好已经买过竞品或正在考虑买），进行30分钟的深度访谈。问题包括：

1. 你为什么想要学这个/买这个？（表面需求）
2. 为什么现在才想要？之前为什么没想？（触发点）
3. 你现在有什么困扰吗？（深层痛点）
4. 如果你买了这个，最希望改变什么？（期待的结果）
5. 是什么阻止你现在就买？（购买障碍）

从5-10次访谈中，你会发现一些共同的主题。这些主题就是你的营销着力点。

## 避坑指南

**误区1：假想的定位而不是真实的定位**

陷阱：你自己觉得你的产品应该吸引"所有想学Excel的25-55岁的人"。但实际上，也许你真正吸引到的是"28-35岁的职场女性"。如果你按照你的假想定位去营销（用男性视角、用大叔语气），反而会把真正的客户群推开。

避坑方法：不要坐在办公室凭空想象。要看实际数据。谁真的在买你的产品？他们有什么共同特征？从真实的客户反推定位，而不是从假想推向客户。具体做法是，找你已有的客户，问一个问题："为什么你选择了我而不是竞品？"他们的回答，往往会揭示你真正的竞争优势和目标人群。

**误区2：竞品分析沦为抄袭**

陷阱：你研究了竞品，然后就复制他们的营销方式。"他在抖音上火了，我也去抖音"、"他用这个文案获得100万阅读，我也用这个文案"。但因为你的品牌、背景、定位都不同，直接复制往往不会有同样的效果。结果是浪费时间和金钱。

避坑方法：竞品分析的目的是"理解"而不是"抄袭"。关键问题是："他们为什么这样做？他们的目标客户是谁？他们的优势是什么？"然后反思："在我的场景下，我应该怎样应用这个思路？"比如，竞品用了"励志故事"的营销方式。你不应该抄他的故事，而应该想"故事营销对我的目标客户是否有效？如果有效，我应该用什么样的故事（可能是你自己的故事）？"

**误区3：故事讲得太虚**

陷阱：你讲的故事，听起来很励志但不真实。"我曾经连Excel都不会，现在我是公司CEO、年薪500万"——这样的故事如果真假掺半，用户会产生怀疑。甚至如果完全是编的，一旦被发现，信任度会彻底崩塌。

避坑方法：用真实的、可验证的故事。最好能给出具体的证据：数字、截图、客户评价、媒体报道等。比如，不说"我帮助了很多学生升职"，而说"我帮助的500名学生中，80%在学完3个月内升职或加薪，其中30人升职到了经理级"，并附上其中几个学生的转介评价。真实的故事，即便不那么跌宕起伏（"我从贫困到富有"），也比虚假的故事更有说服力。因为用户能感受到这是真的。

**误区4：只优化转化，忽视吸引**

陷阱：你的销售页面写得非常精彩，转化率很高（比如30%）。但是到达这个页面的人太少了。结果整体的成交数还是很低。

避坑方法：要平衡"吸引"和"转化"两个方面。好的营销既要有足够的"流量入口"（做好曝光和触达），也要有高效的"转化漏斗"（做好销售页面和电销）。不要把所有资源都用在优化转化率上，而忽视了流量的规模。
