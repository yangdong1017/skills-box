# 第四章 市场分析与竞品研究——看清战场的三个维度

## 开篇：为什么你的产品总是卖不动

有这样一个现象，我接触过的创业者中，大约有70%的人在项目启动前都没有真正做过市场分析。他们的逻辑往往是这样的："我觉得这个产品很好，我需要它，那肯定有很多人也需要。来，我们立刻开始。"

结果呢？花了三个月开发产品，投了十万块做广告，最后发现市场需求远没有想象那么大，或者有强大的竞争对手已经把市场瓜分完了。有些人甚至做出来的产品，竞品早就做过一遍还失败了。

这个故事反复上演，背后的原因就是一个：没有看清战场。

在古代打仗，将军上阵前都要派斥候去侦查。敌人有多少兵力？占据什么位置？有什么弱点？这些信息能决定采用什么战术，甚至决定是否值得一战。做生意也是一样的道理。如果你不知道这个行业有多大、竞争对手有多强、用户到底在哪里，那你的所有营销努力可能都是南辕北辙。

这就是为什么我把"市场分析与竞品研究"放在这一章来讲。在前面三章里，我们重塑了你对流量的认知。从第四章开始，我们要进入"知己知彼"的实战阶段。而"知彼"的第一步，就是彻底理解这个行业和市场。

有人问我，"我没有行业经验，我能做好市场分析吗？"答案是能的。但前提是你得用对方法，花足够的时间，以及最重要的——坚持做到底。

很多人做市场分析就像走马观花，浏览几篇文章，看看竞品的网站，就觉得自己懂行业了。我把这种状态叫"第二阶段停留综合征"。实际上，深度的市场分析需要三个阶段，而大多数人恰好在第二个阶段就卡住了。

## 市场分析的四个核心维度

说到市场分析，很多人觉得这是一件很复杂、很专业的事情。其实不然。市场分析的核心就是四个维度，缺少任何一个，你的分析都是不完整的。

### 市场分析三步走流程图

在深入讲解之前，先看一个完整的市场分析流程。这个流程将指导你如何系统地进行市场调研：

```
┌─────────────────────────────────────────────────────────────────┐
│                    市场分析的系统流程                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  第一步：行业调研                                                │
│  ┌──────────────────────────────────────┐                       │
│  │• 下载行业报告（艾瑞、易观、36氪等）  │                       │
│  │• 查看行业规模、增长率、主要玩家      │                       │
│  │• 了解行业的发展阶段                  │                       │
│  │• 识别行业的关键趋势                  │                       │
│  └──────────────────────────────────────┘                       │
│                     ↓                                            │
│  第二步：竞品分析                                                │
│  ┌──────────────────────────────────────┐                       │
│  │• 列出20个主要竞品                    │                       │
│  │• 分析竞品的定位、渠道、定价          │                       │
│  │• 研究竞品的优势和劣势                │                       │
│  │• 制作竞品对比表                      │                       │
│  └──────────────────────────────────────┘                       │
│                     ↓                                            │
│  第三步：差异化定位                                              │
│  ┌──────────────────────────────────────┐                       │
│  │• 识别市场的细分机会                  │                       │
│  │• 发现竞品忽视的用户群体              │                       │
│  │• 定义自己的独特价值主张              │                       │
│  │• 制定进入策略                        │                       │
│  └──────────────────────────────────────┘                       │
│                     ↓                                            │
│  输出：完整的市场定位和切入点                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**第一个维度，是研究行业的头部品牌如何从零开始的。** 这很重要。因为现在的头部品牌之所以是头部，一定有他们独特的做法。但很多人只看现在的结果——"这个品牌现在多牛，销售额多少"——而忽视了他们当年是怎么起步的。

#### 竞品分析维度对比表

在分析竞品时，需要从多个维度进行对标。这个表格展示了如何系统地对比竞品：

| 分析维度 | 竞品A | 竞品B | 竞品C | 我们的定位 |
|--------|-------|-------|-------|---------|
| **产品定位** | 高端品质 | 大众便宜 | 设计感 | ? |
| **目标客户** | 中产女性 | 学生上班族 | 年轻白领 | ? |
| **价格区间** | 200-500元 | 50-100元 | 150-300元 | ? |
| **主要优势** | 品质+口碑 | 价格+便利 | 设计+品牌 | ? |
| **主要劣势** | 价格偏高 | 质量参差 | 认知度低 | ? |
| **主要渠道** | 天猫+小红书 | 淘宝+拼多多 | 小红书+抖音 | ? |
| **获客方式** | KOL推荐 | 低价活动 | 内容营销 | ? |
| **客户满意度** | 4.8星 | 4.2星 | 4.5星 | ? |
| **市场份额** | 15% | 20% | 8% | ? |
| **融资状况** | B轮已融 | A轮融资 | 天使轮 | ? |

这个表格的关键作用是帮你**看清竞品的布局逻辑**。你会发现：
- 高端品牌都在强调品质和口碑
- 便宜品牌都在强调价格和渠道覆盖
- 新兴品牌都在做内容和社交营销

这些规律，就是你制定自己策略的参考。

我举个例子。国内有一个很知名的内衣品牌，现在年销售额十几个亿。但如果你去研究这个品牌的初期，你会发现他们最早是怎么做的？直播卖货。那时候直播在很多人眼里还不太正规，但这个品牌通过直播获得了第一批用户和口碑。这个信息对你意味着什么？如果你现在要做内衣，直播可能就应该是你的核心渠道之一。而不是跟着他们现在的做法——去铺设线下店铺。

深挖品牌初期的故事，你会发现很多被遗忘的营销线索。那个时候，他们的预算往往很有限，所以必须找到最高效的获客方式。这些高效的方式，往往比他们现在花大钱的方式更有借鉴价值。你可以问自己几个具体的问题：这个品牌第一个代理商是从哪里找到的？第一批顾客是从哪个平台来的？他们最初是怎么定价的？是一开始就做品牌，还是从代工开始？这些答案，决定了你应该怎么起步。

**第二个维度，是研究红海市场中的机会。** 很多新手听说某个行业是红海，就直接放弃了。其实这是错误的。红海不是没有钱赚，红海里已有的大品牌就是最好的教材。他们为什么能在红海里活得好？因为他们找到了差异化。

比如说袜子市场。袜子市场显然是红海，各种大品牌早就占据了。但如果你仔细研究，会发现大品牌主要聚焦在基础款和大众需求。那细分呢？彩色袜子、毛绒袜子、卡通袜子——这些细分品类需求量就没那么大，但竞争也小得多。有一个创业者就是通过专做"潮牌袜子"这一个细分品类，硬生生在红海里杀出了一条血路。

这就是红海中的"差异化切口"。研究红海，你要问自己这样的问题：这个市场的大品牌做得有什么遗漏的地方？有没有某个用户群体被忽视了？有没有某个使用场景没有被照顾到？比如说，袜子市场有没有人在关心"办公白领穿的袜子"或"户外爱好者的袜子"？如果有，那就是你的切入点。红海看起来一片血红，其实如果你足够细致，会发现很多被忽视的缝隙。这些缝隙，就是你的蓝海。

**第三个维度，是寻找蓝海——真正的空白市场。** 蓝海很吸引人，因为竞争少。但蓝海的问题是，市场容量往往也比较小。所以蓝海的打法应该是"先在蓝海做成头部，然后慢慢扩展到邻近的细分"。而不是一味地追求蓝海。

我见过很多创业者，他们发现了一个完全没人做的蓝海领域，很兴奋地投入进去。结果三个月后发现，这个蓝海之所以没人做，不是因为没机会，而是因为这个市场太小了——每个月的总需求可能才几百块钱。这样的蓝海，做得再好也赚不到钱。所以，蓝海要选"刚好小但有成长空间"的市场，而不是"太小根本没有市场"的市场。怎么区分？很简单，看有没有相关的搜索量。如果月搜索量在1000以上，说明这个需求虽然小众但真实存在。如果搜索量在100以下，那就要谨慎了。

**第四个维度，是研究商业模式——这个行业到底怎么赚钱的。** 这是最多人忽略的。很多人看到一个产品畅销，就以为做这个产品就能赚钱。其实错了。同一个产品，不同的公司采用不同的商业模式，盈利能力可能完全不同。

有人通过内容+电商的模式赚钱，有人通过代理+分销的模式赚钱，有人通过直播带货的模式赚钱。同样的产品，商业模式不同，对应的渠道、定价、人员配置都完全不同。如果你没有理解这个行业的主流商业模式，你的所有决策都可能是错的。

比如说，服装行业看起来是单一的"卖衣服"，但实际上有很多种商业模式。有的品牌通过跟风快反的方式赚钱（ZARA模式），有的通过高端定位和品牌力赚钱（LV模式），有的通过社交电商和私域粉丝赚钱（李子柒模式），有的通过直播带货赚钱。每种模式对应的供应链、定价、营销都完全不同。如果你选错了模式，再好的产品都救不了你。所以，研究商业模式时，你要看这个行业赚钱的第一梯队品牌，他们都在用什么模式。他们的利润率是多少？都在哪些渠道销售？消费者是怎么找到他们的？这些数字，是你制定自己商业模式的参考。

这四个维度相互关联，形成一个闭环。研究了头部品牌的起步，你知道了"机会在哪"——他们当时用了什么渠道、怎么获得第一批用户、第一个产品是什么。研究了红海的差异化和蓝海的细分，你知道了"自己该往哪个方向走"——是跟风做大众市场，还是精细化做细分市场。研究了商业模式，你知道了"自己能不能赚到钱"——这个模式的利润空间有多大，需要多大的初期投入。

有一个实际的案例可以说明这四个维度的作用。假如你想做"宠物零食"生意。第一步，研究头部品牌的初期是怎么做的——比如某品牌最初通过小红书的宠物博主推广起家。第二步，研究红海里的差异化——大品牌都在做"通用型零食"，但有没有针对"过敏宠物"的零食？有没有针对"老年犬"的零食？第三步，寻找蓝海——比如发现"狗狗牙齿护理零食"这个细分还没有专业品牌。第四步，研究商业模式——了解到这个品类的头部品牌，70%销售来自抖音直播，定价是3-5倍成本。有了这四个维度的认知，你就能清晰地知道自己该做什么。

### SWOT分析四象限

市场分析的最后一步是SWOT分析，帮你理清自己的竞争位置。这是一个四象限模型：

```
┌─────────────────────────────────────────────────────────┐
│                    SWOT 分析矩阵                         │
├──────────────────────┬──────────────────────────────────┤
│                      │                                  │
│  S（优势）           │  W（劣势）                       │
│  ─────────────        │  ─────────────                  │
│  • 我们有什么独特     │  • 我们缺少什么                │
│    的优势？           │  • 和竞品相比有何不足？        │
│  • 技术/成本/         │  • 我们的瓶颈在哪?             │
│    渠道优势           │  • 如何快速补齐？              │
│                      │                                  │
├──────────────────────┼──────────────────────────────────┤
│                      │                                  │
│  O（机会）           │  T（威胁）                       │
│  ─────────────        │  ─────────────                  │
│  • 市场有什么         │  • 有什么事可能伤害我们？      │
│    增长机会？         │  • 竞争对手可能怎样攻击我们？  │
│  • 用户需求变化      │  • 政策/技术/行业变化的风险  │
│  • 新的渠道/合作     │  • 如何提前防范？              │
│                      │                                  │
└──────────────────────┴──────────────────────────────────┘

示例：以在线教育品牌为例

┌─────────────────────────────────────────────────────────┐
│                    SWOT 分析示例                        │
├──────────────────────┬──────────────────────────────────┤
│  S（优势）           │  W（劣势）                       │
│  • 创始人有名校背景   │  • 资金规模小于竞品             │
│  • 课程内容原创性强   │  • 品牌知名度低                │
│  • 学员完课率高       │  • 销售团队经验不足             │
│  • 成本结构优化       │  • 线下渠道布局晚               │
├──────────────────────┼──────────────────────────────────┤
│  O（机会）           │  T（威胁）                       │
│  • 在线教育增长30%    │  • 头部品牌强势占领市场        │
│  • 新增用户对品质     │  • 竞品融资充足，砸钱推广     │
│    有需求             │  • 政策监管趋严                │
│  • 短视频带货兴起     │  • 用户获客成本上升             │
│  • 国外教育方法输入   │  • 技术迭代淘汰老课程          │
└──────────────────────┴──────────────────────────────────┘
```

**做完SWOT后的决策逻辑：**
- **S+O组合**：用优势抓住机会 ← 这是你应该重点投入的方向
- **S+T组合**：用优势对抗威胁 ← 这是防守的重点
- **W+O组合**：通过改进弱点来抓住机会 ← 需要补齐短板
- **W+T组合**：这是危险信号 ← 要提前规避

## 市场分析的三个阶段：从混乱到感觉到系统

真正深度地研究一个行业，需要三个阶段。这个过程不能跳跃，也不能加速。每个阶段都很关键。

**第一阶段，是信息的混乱阶段。** 当你开始下载行业报告、查阅竞品数据、阅读相关文章的时候，你会发现信息爆炸。行业报告说这个方向很有前景，竞品说他们的优势在这，用户反馈说他们最关心那。各种声音混杂在一起，如果你的脑子还没有框架，就会感到晕头转向。

这个阶段其实是很有价值的，虽然看起来很混乱。因为你在接收全方位的信息。但问题是，很多人在这个阶段就放弃了，或者被动地停留在这里。他们下载了一堆文件，标记了一些，但从来没有系统地整理过。有人会说："我已经看了50份行业报告，我肯定懂这个行业了。"但实际上，如果他们没有把这50份报告的信息提炼成10个核心观点，那他们其实还是在混乱阶段。

**第二阶段，是形成感觉的阶段。** 经过几天甚至几周的阅读和思考，你开始有感觉了。你能凭直觉说出这个行业的大概情况："哦，这个行业现在最热的是直播卖货"、"最大的问题是竞争太激烈"、"好像有个蓝海机会在细分品类"。

这个阶段的感觉大概有70-80%是对的。你已经接触了足够的信息，能够形成一些初步的判断。但这种感觉很危险，因为它给你一种"已经懂了"的幻觉。有很多聪明的创业者就在这个阶段停留下来，觉得"大体了解了，可以开始干了"。他们会说："我已经看出来了，这个行业最关键的是内容和直播结合，我就按这个方向做。"结果呢？往往会栽跟头。因为他们的理解还是片面的、不系统的。这是最致命的陷阱。

**第三阶段，才是真正的分类和归纳。** 你需要把所有收集到的信息，用结构化的方式整理成表格或脑图。比如列出这个行业的所有主要品牌、他们各自的定位、商业模式、目标人群、销售渠道、价格带。然后横向对比，看看有什么规律。

这个阶段看起来最枯燥，但是最有价值的。在这个过程中，你不是在消费信息，而是在加工信息。你问自己："为什么品牌A选择了直播，品牌B选择了小红书？""为什么他们的定价差异这么大？""为什么销量最高的品牌不一定利润最高？"这些问题的答案，才是真正的市场认知。

只有到了这一阶段，你才能说自己真正懂了这个行业。因为你对信息做了二次加工，而不仅仅是看过。你能回答具体的问题，而不仅仅是说出模糊的感觉。

我建议的做法是这样的：拿出一周的时间，每天花6-8小时专注地研究。不要同时做其他事情，这样会分散注意力。

第一、二天，大量下载和阅读报告。下载渠道包括：艾瑞、易观、中商产业研究院、36氪研究院等。一天至少读3-5份报告，做笔记记录关键数字和观点。

第三、四天，列出行业的20个主要品牌和竞品。不是随意列，而是按照销量、口碑、融资额等维度来选。然后对每个品牌做初步分析：他们怎么开始的、现在在哪些渠道销售、定价多少、目标人群是谁。

第五、六天，制作汇总表格。表格应该包括这8列：品牌名、定位、商业模式、主渠道、客单价、月销量、融资轮次、目标人群。填数据的时候要多渠道验证，一个数据来源不够，要看2-3个来源。

第七天，反复看这个表格，找出规律。比如：定价高的品牌都强调什么？销量最高的品牌有什么共同特征？新兴品牌和老牌品牌的区别在哪？形成自己的认知框架。

这一周下来，你对这个行业的理解深度会远远超过那些"浏览式"学习的人。你能回答具体的问题，也能预测这个行业的发展方向。

## 市场调研的四个实操维度

理论讲到这里，现在进入实战。在实际操作中，市场调研需要从四个维度展开。这四个维度的顺序很重要，不能颠倒，也不能跳过。

**维度一：找可模仿的竞品和商业模式。** 你需要研究，这个行业中有哪些公司做得很成功。他们的商业模式是什么？他们的定价是多少？他们的主要销售渠道是什么？他们的目标人群是谁？

这不是抄袭，而是学习。古人说"他山之石可以攻玉"，同样的道理。竞品的成功案例就是你最好的教科书，告诉你什么是可行的。具体的做法是：在淘宝、天猫、抖音、小红书等主流平台上，搜索你的行业关键词。按销量排序，看销量最高的前20个品牌。对每个品牌做一个完整的页面截图，包括他们的产品图、文案、价格、评价等。然后问自己：这个品牌为什么这样设计页面？为什么他们强调这个点而不是那个点？为什么价格是这个数字？这样逆向思考，你就能理解他们的策略。

**维度二：找自己的差异化竞争力。** 在研究了竞品之后，你需要问自己：我和他们有什么不同？我有什么优势是他们没有的？这涉及到一个方法论叫"卡诺模型"。简单说，就是把产品的特性分为三类：基础特性（所有竞品都有）、期望特性（用户会比较的）、兴奋特性（能让用户惊喜的）。你的优势应该在期望特性或兴奋特性上，否则就没有差异化。

举个例子。如果你要做蜂蜜生意，"天然蜂蜜"就是基础特性（所有品牌都说自己是天然的）。"来自某个特定地区"是期望特性（用户会比较不同产地的蜂蜜）。"有独特的风味和故事"就是兴奋特性（让用户觉得这不仅仅是食品，而是一个故事）。所以，你的竞争力应该在后两个维度。很多品牌失败，就是因为他们只强调"天然、纯正"这样的基础特性，和所有竞品一样。

**维度三：深入了解目标客户群体。** 这需要你做一个大胆的动作：假装成一个客户，去体验竞品和这个行业。进入目标客户聚集的社群、论坛、讨论组。不是为了营销，而是为了听他们说话。他们在抱怨什么？他们关心什么？他们需要什么样的解决方案？

很多市场调研的报告都是冷冰冰的数字。但最宝贵的信息，往往来自于和3-5个真实的目标客户的深度对话。比如说，你要做母婴产品。不要只看问卷数据，要进入亲子论坛、妈妈微信群、小红书宝妈账号，真实地观察和交流。你会发现，妈妈们最关心的往往不是产品本身的功能（这个大家都知道），而是"有没有其他妈妈用过""效果怎么样""有没有副作用"这样的真实使用反馈。这个认知，会直接改变你的营销策略。

**维度四：确定客户聚集的地方。** 这是最后一步，也是最实用的一步。你需要知道目标客户主要在哪些平台和场景聚集。他们是在淘宝找产品，还是在抖音看推荐？他们是在微信社群讨论，还是在知乎提问？不同的聚集地对应不同的营销策略。

关键是，不同的平台代表用户处于不同的决策阶段。在抖音上看推荐的用户，往往是还没有明确需求的"浏览阶段"。在知乎提问的用户，往往是已经有明确问题的"主动搜索阶段"。在微信群讨论的用户，往往是"比较决策阶段"。每个阶段的用户，需要的信息和营销方式都不同。所以，你必须既要知道客户在哪些地方，更要知道客户在不同地方处于什么样的决策阶段。

这四个维度缺一不可。很多人只做了第一个维度（看竞品），就启动项目。结果没有思考过差异化，没有深入接触过客户，也没有找到客户最聚集的地方。营销再好也很难成功。

## 竞品研究的完整流程

说到竞品研究，我需要特别强调一个概念：竞品的定义要广。

很多人认为竞品就是"做同样产品的公司"。这是错的。竞品应该这样定义：所有能解决客户同一需求的解决方案，都是你的竞品。

比如说，一个人想"放松身心"这个需求。解决这个需求的手段可以是按摩、瑜伽课、冥想APP、到咖啡馆坐一会，甚至看一部电影。这些看起来完全不同的"产品"，实际上都在竞争这个人的时间和金钱。

再比如说，如果你要做"在线教育"的生意，你的竞品不仅仅是其他的在线教育平台，还包括书籍、线下培训班、YouTube上的免费视频、朋友的建议等等。所有这些能帮助用户学习的方式，都是你的潜在竞品。理解了这一点，你才能发现真正的竞争对手，也才能找到真正的差异化机会。

基于这个理解，竞品调研的标准流程是这样的：

**第一步：体验竞品的销售流程。** 打开百度或抖音，搜索你要做的产品或服务的关键词。进入竞品的销售页面（网站、小程序、店铺），完整地走一遍他们的销售流程。看看他们怎么介绍产品的，怎么讲卖点的，怎么处理客户的异议的。

这个过程中，记录下来他们强调的三个最大的卖点是什么。往往他们强调的点，就是客户最关心的点。同时也要注意，他们在哪个环节放置了降低购买阻力的信息，比如"30天无理由退款"、"已有10万人使用"这样的表述。

**第二步：分析竞品的官网和描述页。** 进入竞品的官方网站，仔细看他们的首页和产品描述页。特别注意这几点：
- 标题怎么写的？是强调功能（"高效率管理工具"），还是强调结果（"帮你每年节省100小时"），还是强调身份（"为专业人士设计"）？
- 首图是什么？是产品的样子，还是使用场景，还是用户的成果展示？
- 主要讲了什么功能或利益？列出来对比，看他们强调哪3-5个核心点。
- 定价是多少？有没有多个价格梯度？
- 有什么保证或承诺？30天退款、一年保修、还是终身会员？

这些都不是随意设计的，每一个设计决策背后都代表了他们对市场的理解和对消费者心理的把握。

**第三步：用企查查找到竞品的联系方式，主动打电话。** 这步很关键，也是很多人不敢做或不想做的。打电话到竞品公司，以客户的身份咨询他们的产品和服务。

具体怎么做？你可以说"我想了解你们的产品，但不确定是否适合我，能给我讲讲吗？"这样既能获得信息，又不会暴露你的真实身份。在这个过程中，你会听到他们销售人员怎么推销的（他们会强调什么），他们如何回答客户的疑虑的（常见的异议有哪些），他们怎么促成交易的（用什么技巧关单）。这比看网站获得的信息更鲜活、更真实。

为了保护隐私，建议用临时手机号或朋友的号码。同时也要做好记录，最好在通话中间或结束后立刻记下关键信息，因为电话信息容易遗忘。

**第四步：用5118等工具挖掘需求关键词。** 5118是一个关键词分析工具（网址是5118.com）。你可以输入行业的主关键词，工具会列出所有相关的搜索词、搜索量、长尾词、疑问词等。

这个数据太宝贵了。因为搜索词代表的是真实的、明确的用户需求。通过分析这些词，你能知道用户到底在关心什么。比如说，你搜索"减肥产品"，5118会列出各种长尾词，比如"快速减肥""安全减肥""健康减肥"等。不同的长尾词代表不同的用户心理和需求。通过这个，你能了解这个行业用户最关心的痛点排序。

**第五步：整理成竞品对比表。** 把收集到的所有信息，整理成一张表格。包括品牌名、主要卖点、定价、销售渠道、目标人群、搜索量排名等。横向对比，看看有什么规律。

这个表格很重要，它帮助你从零散的信息中看出整体的市场结构。你会发现，定价高的品牌都强调什么。销量高的品牌在哪些渠道最活跃。新兴品牌和老牌品牌的差异在哪里。这些规律，就是你制定自己策略的依据。

## 从搜索需求到商业机会

这是我特别想讲的一个话题，因为它直接关系到你能不能发现真正的商业机会。

每个搜索词背后，都代表一个真实的需求。而且，搜索词的多样性本身，就是市场的多样性。需要注意的是，不同类型的搜索词代表了用户的不同需求阶段。

比如说，"袜子"这个关键词，通过5118可以扩展出几百个长尾词。其中不仅有"运动袜子"、"儿童袜子"这样的品类词，还有"袜子发黄怎么办"、"袜子起球怎么处理"这样的问题词。还有"袜子和鞋哪个先买"这样的决策词。这些不同类型的词代表了不同的用户心理：

- 品类词（"运动袜子"）代表用户有明确的品类需求
- 功能词（"防臭袜子"）代表用户关心的产品属性
- 问题词（"袜子发黄怎么办"）代表用户遇到了实际的困扰
- 比较词（"什么袜子最舒服"）代表用户在决策阶段
- 长尾词（"30多岁女性穿什么袜子"）代表非常细分的需求

有一个创业者就是通过分析这些问题词，发现了一个机会：做一个"袜子护理和搭配指南"的公众号。这个公众号通过输出有价值的内容（怎么选袜子、怎么搭配、怎么护理、袜子怎么快速变白），吸引了大量目标用户。后来通过在公众号中推荐好袜子品牌，获得了返佣。这个账号现在每个月能通过返佣赚5-10万块。

这个例子说明了什么？从一个搜索词，你可以延伸出无数个商业方向。问题词可以延伸成内容业务、教育业务；品类词可以延伸成专业的细分品牌；比较词可以延伸成导购或评测业务；甚至身份词（"25岁白领""宝妈"）可以延伸成社群业务。

关键是，你要学会看穿搜索词背后的真实需求，而不仅仅是看这个词的搜索量。搜索量高的词往往竞争也激烈，但搜索量中等、竞争少的长尾词，往往才是最容易突破的地方。

## 市场数据的收集和应用

讲到这里，有人会问："数据去哪里找？我得全部自己调研吗？成本是不是很高？"

答案是：免费的数据源就够用了。你不需要花大钱买数据，也不需要自己做问卷调查。免费的渠道有很多，关键是知道怎么用。

**第一个来源是行业报告。** 在百度搜索框搜索你的行业名称+报告，比如"直播带货行业报告"，就能找到一堆。很多权威的市场研究机构（如艾瑞、易观等）会定期发布免费的行业报告。这些报告里有市场规模、增长率、主要玩家等信息。你需要学会甄别报告的质量——看发布机构是否权威、数据时间是否最新、数据是否有引用来源。一份好的报告能为你节省几个月的调研时间。

**第二个来源是电商平台。** 淘宝、京东、拼多多、抖音小店等，都是天然的市场数据库。在这些平台上搜索你的产品关键词，按销量排序。销量最高的产品是什么？价格多少？评价怎么样？有多少人购买？这些都是实实在在的市场信息。你可以用一个简单的方法来估算月销额：月销量×价格。这样你就能知道这个品类每个月的市场规模。比如说，搜索"防晒霜"，销量最高的产品月销5万件，平均价格100块，那么这个品类每个月的市场规模就是500万块。如果你能拿到1%的市场份额，那就是5万块月收入。

**第三个来源是专业的数据平台。** 比如馋妈妈，可以查看抖音电商的销售数据；搜狗，可以查看微信公众号的排名；5118，可以查看关键词的搜索趋势。这些工具虽然不是免费的，但提供的数据价值很高。有些工具还提供免费的基础版本。我建议的做法是，先用免费版本验证你的想法是否正确，如果数据看起来有价值，再考虑付费获取更详细的数据。

**第四个来源是社交媒体。** 微博、小红书、抖音上的用户评论和讨论，都是真实的市场反馈。看看用户在讨论什么品牌，他们最关心什么属性，他们的吐槽是什么。这些都是比报告更新鲜、更真实的数据。比如说，在小红书上搜索"纸尿裤评测"，你会看到大量用户的真实使用反馈。哪个品牌最被吐槽？吐槽的原因是什么？这对你的产品设计、定价、定位的影响，往往比任何报告都大。

收集数据的原则是：不要只依赖一个来源。多个来源交叉验证，才能确保数据的准确性。同时，不要被数据淹没。你的目标是提炼出最关键的20个品牌、最重要的10个指标、最常见的5个用户痛点。这样才是有用的数据，否则数据越多越容易迷茫。

## 行业数据汇总表：把信息变成认知

所有的信息收集，最终都应该汇总成一个结构化的表格。这个表格，我叫它"行业数据汇总表"。

这个表格应该包含8列：品牌名称、赚钱模式、发展历程、主要渠道、产品定位、价格带、销售数据、目标人群。

举个例子。假如你要做袜子生意，你的表格可能是这样的：

| 品牌 | 赚钱模式 | 发展历程 | 主要渠道 | 产品定位 | 价格 | 月销量 | 目标人群 |
|------|--------|--------|--------|--------|-----|--------|--------|
| 品牌A | 淘宝+直播 | 先做供应商，后做品牌 | 淘宝+抖音 | 基础款高性价比 | 5-15元 | 10万+ | 大众消费 |
| 品牌B | 微商+社群 | 从朋友圈起家 | 微信社群 | 潮牌+私密定制 | 20-50元 | 1万+ | 年轻白领 |

当你列出20个品牌的汇总表后，你看着这张表，自然而然就能看出规律。你会发现：
- 高价格的品牌往往强调设计感或品质；
- 高销量的品牌大多在淘宝和抖音；
- 小众品牌往往通过社群和口碑获客；
- 新兴品牌大多采用直播或内容营销。

这张表，就是你对这个行业的完整认知。而且这个认知是有数据支持的，不是凭空想象。

## 产品定位的三个要素

讲了这么多关于市场和竞品的分析，最后还要讲一个关键点：你的产品应该怎么定位。

市场分析的最终目的，就是为了给自己的产品找到清晰的定位。而产品定位，有三个关键要素，它们就像一个三脚架，缺少任何一条腿都会倒下。

**第一个要素是包装和视觉表现。** 产品的图片怎么拍、文案怎么写、风格怎么设计，都会影响客户对产品的认知。同样一个杯子，用白底纯图片展示和用场景图展示，给人的感觉完全不同。前者显得廉价，后者显得精致。这不是欺骗，而是呈现方式的问题。一个高品质的产品，如果包装得不好，客户永远看不到它的价值。相反，一个中等产品，如果包装和视觉设计做得好，能卖出高端产品的价格。

具体来说，包装包括五个方面：产品摄影（照片质感）、文案描述（怎么讲卖点）、设计风格（高端 vs 亲民）、品牌故事（有没有故事背景）、用户见证（有没有用过的人评价）。如果你是新品牌，没有用户见证，可以用对标品牌的评价，或者自己先用再评价。

**第二个要素是定价。** 定价本身就是一种信号。价格高意味着这是高端产品，价格低意味着这是性价比产品。定价决定了你吸引什么样的目标客户，也决定了你的利润空间。关键是，定价要有逻辑，而不是凭感觉。你应该研究竞品定价，然后根据自己的成本和目标利润率来定价。同时也要考虑心理学因素：99块比100块看起来便宜，这是心理学效应。299块的定价看起来比300块更有吸引力，即使只差1块钱。

**第三个要素是产品功能或服务形式。** 课程要包含什么内容？产品要有什么功能？服务要提供什么承诺？这些都需要基于你对市场和客户的理解来设定。比如说，如果你发现你的目标客户最关心"有没有退款保证"，那你的产品承诺就应该强调"30天无条件退款"。如果你发现目标客户最关心"能不能快速看到效果"，那你的服务形式就应该设计得让用户能快速获得初步成果，而不是让他们等到最后才看到成果。

这三个要素，缺少任何一个都不行。很多创业者只关注功能（我的产品质量好），忽视了包装（但看起来很廉价）和定价（定价没有逻辑），结果产品再好也卖不出去。

这里有一个真实的案例：有两个创业者都做"时间管理APP"。产品功能其实很接近，都是任务管理+提醒+数据统计。但一个定价99块钱、简约的包装风格、针对"忙碌的管理者"。另一个定价29块钱、花里胡哨的设计、针对"所有想提高效率的人"。结果第一个每个月能卖到3-5万块，而第二个每个月也才能卖到3-5万块。看起来销量差不多，但利润完全不同。这就是定位和包装的力量。

| **搜索需求** | 搜索引擎 | 5118、爱站网 | 易 | 周更新 | 找关键词、了解用户需求 |
| **融资信息** | 企业数据库 | 企查查、启信宝 | 易 | 实时 | 了解竞品融资进展、团队规模 |
| **舆论声量** | 社交媒体 | 小红书、知乎、微博 | 易 | 实时 | 了解用户评价、发现问题 |
| **内容表现** | 内容平台 | 新榜、抖大助手 | 中 | 日更新 | 分析竞品的内容策略 |
| **微商数据** | 社交电商 | 小蜜蜂、袋鼠 | 中 | 周更新 | 了解微商团队规模和业绩 |
| **店铺运营** | 官方后台 | 生意参谋（淘宝） | 中 | 实时 | 精准数据（需是商家） |
| **流量数据** | 第三方监测 | SimilarWeb、Alexa | 中 | 月更新 | 网站/APP的流量趋势 |
| **消费者反馈** | 评价系统 | 淘宝评价、大众点评 | 易 | 实时 | 了解产品问题和改进方向 |
| **竞品信息** | 公司官网 | 竞品网站 | 易 | 不定 | 了解品牌故事、产品体系 |

**实用建议：**
- 初期研究可以全用免费渠道（淘宝、5118基础版、知乎）
- 深度运营后可以考虑付费工具（5118高级版、新榜等）
- 不要花大钱一次性购买所有工具，而是根据实际需求逐步添加

## 市场分析工具和平台的实用指南

前面讲了那么多方法论，这里再讲一下具体要用的工具。不是要你全部都用，而是根据你的行业来选择最合适的工具。

**搜索数据工具：** 5118、爱站网。这两个都能查关键词的搜索量、竞争度、长尾词、相关词。5118的优势是词库全、数据更新快；爱站的优势是界面简洁、免费功能够用。

**电商数据工具：** 生意参谋（淘宝官方）、抖音电商后台、京东商智。如果你要在电商平台销售，这些官方工具能给你最准确的数据。不过这些都需要是商家才能用。对于初期研究，你可以用第三方的馋妈妈、抖音数据分析工具等。

**竞品分析工具：** 企查查（公司信息）、启信宝（融资信息）、小蜜蜂（微商数据）、新榜（内容账号数据）。根据你的行业选择，比如做知识付费就用新榜，做微商就用小蜜蜂。

**社交媒体工具：** 微博搜索、小红书搜索、知乎搜索。这些平台的搜索功能本身就是最强大的调研工具。你可以直接搜索相关话题和关键词，看用户的真实讨论。

选工具的原则是：先用免费版本，验证你的想法是否有价值。如果数据确实有指导意义，再考虑付费获取更详细的信息。不要一开始就花大钱买各种工具，结果用不了两个月就闲置了。

## 本章小结

深度的市场分析，就是从混乱的信息中，提炼出清晰的认知。这个过程分三个阶段：收集信息的混乱阶段、形成感觉的阶段、系统整理的阶段。很多人在第二阶段就停留了，觉得"已经有感觉了可以干了"，这是最致命的陷阱。真正的认知，只有在第三阶段系统整理之后才能产生。

市场分析的四个核心维度是：研究头部品牌的起步（了解这个行业是怎么发源的）、研究红海的差异化（寻找大市场中的细分机会）、寻找蓝海的细分（发现真正的空白但有需求的市场）、理解商业模式（搞清楚这个行业怎么赚钱）。四个维度结合，才能给你完整的市场认知。

在实操层面，需要做四个维度的调研：找可模仿的竞品和他们的成功秘密、找自己的差异化竞争力（卡诺模型）、深入了解目标客户的真实需求（不是凭想象）、确定客户聚集的地方和聚集的阶段。最后把所有信息汇总成一张表格，这张表格就是你的"市场认知地图"。有了这张地图，你才能清晰地看到这个市场的全景，知道自己该如何定位和切入。

## 操盘手练习

这些练习不是为了完成作业，而是为了给你的项目建立坚实的基础。很多创业者失败，就是因为跳过了这些基础工作。

**练习一：制作你所在行业的数据汇总表**

选择你要做的行业，用一周时间完成这个练习。这一周的投入，可能会为你节省半年的摸索时间。

1. 列出这个行业的20个主要品牌或竞品。怎么找？在淘宝、抖音、小红书、天猫等平台搜索你的行业关键词，按销量排序，从第一个开始记录。同时用5118查询关键词排名，记录排名前20的品牌或内容。

2. 收集每个品牌的8个关键数据：品牌名、赚钱模式（直播/淘宝/社群等）、发展历程（初期怎么起步的）、主要渠道（在哪里销售最多）、产品定位（高端/性价比等）、价格带（多少钱）、销售数据（月销量）、目标人群（谁在买）。数据来源包括：品牌官网、电商页面、5118搜索数据、企查查融资信息、小红书用户评价等。

3. 看着这张表格，反复思考并写出你对这个行业的三个核心认知。比如：这个行业利润最高的品牌和销量最高的品牌不是同一家，原因是什么？这个行业在过去两年有什么变化？新兴品牌和老品牌的主要区别在哪里？

4. 根据这些认知，思考自己的定位应该是什么。比如：我是走高端路线还是性价比路线？我应该专注哪个渠道而不是平均分散精力？我应该瞄准哪个客户群体？

**练习二：做一次完整的竞品调研**

选择这个行业中最成功的一个竞品（销量最高或融资最多的那个），做一次完整的调研。这不是一两个小时能完成的，要花3-5天的时间。

1. 在淘宝或抖音上体验他的销售流程。从搜索关键词开始，看看他们的商品排名为什么这么高。点进去，看产品图怎么拍的、文案怎么写的、价格怎么设置的、评价怎么样的。走一遍完整的购买流程（不一定要买，但要把流程走通）。记录关键节点，比如他们在哪个环节强调了什么卖点，在哪个环节给了什么保证。

2. 分析他们的官网和产品描述。看官网首页的设计、产品页的描述框架、售后说明的承诺。提取他们最核心的卖点。你会发现，他们强调的点往往只有3-5个，不会更多。这说明他们已经研究清楚了用户最关心什么。

3. 用临时手机号打电话咨询。假装自己是一个真实的潜在客户，问他们一些常见的问题。在这个过程中记录他们的销售话术，特别是他们怎么处理你的异议的。他们说的哪些话最有说服力？他们是怎么促成转化的？

4. 用5118查询他们相关的搜索关键词和用户需求。看看用户搜索什么词能找到他们。这些词告诉你用户的真实需求是什么。然后看看这些词的搜索量、竞争度，推断出他们的主要流量来源在哪里。

5. 整理成一份完整的竞品分析报告。包括：基本信息（品牌、定位、渠道）、销售策略（如何定价、如何展示、如何促成）、目标客户（是谁、关心什么）、竞争优势（比同行好在哪里）、可学习的点（你可以借鉴什么）。这份报告就是你项目的"对标分析"。

**练习三：找出行业的三个发展阶段**

选择你的行业，思考这个行业的发展历程。通常一个行业会经历三个阶段：

1. 萌芽期（市场刚开始出现需求）——这个阶段有哪些品牌在做？他们是怎么起步的？

2. 成长期（市场快速扩大，竞争者大量涌入）——这个阶段哪些品牌成功了？他们做对了什么？

3. 成熟期（市场增长放缓，竞争趋向白热化）——这个阶段怎样才能突出重围？

理解了行业的发展阶段，你才能判断自己现在进入这个行业是早还是晚，应该用什么策略。如果行业还在成长期，你可以快速跟风。如果行业已经进入成熟期，你就必须找到差异化的点才能活下去。

## 避坑指南

**误区一：在第二阶段停留，自以为有感觉就够了**

很多人花两三天看了一些文章和报告，就觉得自己"大体懂了"这个行业。这种浮躁的感觉，往往会导致错误的决策。有个创业者就这样，花了5天看了各种报告，觉得"我已经懂这个行业了，开始干吧"。结果呢？三个月投入50万，销售额还不到10万。原因就是他对市场理解不够深，产品定位错了，渠道选择也错了。真正的认知，需要系统的整理和反复的思考。一个星期的深度研究，比一个月的浏览式学习有效得多。关键在于，深度研究能让你发现市场上隐藏的规律，这些规律直接指导你的决策。浏览式学习只能给你模糊的印象。

**误区二：过度相信调研数据，忽视实际验证**

数据很重要，但数据可能被刷单、被优化。有些平台数据可能已经滞后。比如说，抖音上显示某个产品"月销10万+"，但如果你实际去看货架期和评价增速，可能发现这个数字是虚的。最好的做法是"数据+观察+用户反馈"的三角验证。真正去用过产品、和5个真实客户聊过天，比看100张表格更有价值。数据告诉你"是什么"，但只有实际观察和交流才能告诉你"为什么"。

**误区三：看到红海就放弃，不研究其中的机会**

红海不是死路，红海里的差异化机会往往更容易把握。因为你有参考点、有目标、有清晰的竞争对手可以学习。反而蓝海因为太陌生，很多人做不好。比如说，你想做护肤品，护肤品显然是红海。但护肤品中有没有"特定皮质人群"的细分？有没有"特定年龄段"的细分？有没有"特定使用场景"的细分？研究深了，你会发现一堆机会。不要盲目追求蓝海，在主行业内寻找细分机会，成功的概率会高得多，因为市场容量更大。

**误区四：只做研究，不做决策**

有些人陷入了"无限研究"的陷阱。他们觉得"还需要再看一份报告"、"还需要再调查一个竞品"、"还需要再思考一下"。结果永远无法启动项目。市场分析没有100%完美的时刻，什么时候算够了？我的建议是：花一周时间做深度研究，然后就要做决策。后续在实践中继续学习和调整。"80%的信息就可以决策"是创业的必要准则。

**误区五：只看竞品的表面，不研究竞品的底层逻辑**

看到竞品在做直播，就以为"这个行业都应该做直播"。看到竞品定价很高，就以为"高端定位是唯一的出路"。实际上，竞品的做法都是基于他们的资源、团队、时间等因素的。你应该理解竞品为什么这样做，而不是盲目复制。比如说，一个竞品做直播是因为他们有网红资源，但如果你没有，盲目复制直播策略就会失败。正确的做法是：理解竞品的逻辑，然后根据自己的资源做出调整。

**误区六：市场分析完成就觉得万事大吉，忽视了后续的实践验证**

市场分析只是基础，真正的市场认知来自于实践。你在分析阶段的假设，到了实践中可能被推翻。比如说，你分析出来"直播是最佳渠道"，但实际做直播时发现自己不适合做直播，可能小红书会更好。所以，市场分析完成后，还要做"小范围验证"。选择一个最有把握的切入点，小规模试验1-3个月，看实际效果和分析是否匹配。如果匹配，再加大投入。如果不匹配，快速调整。这样的"分析+验证+调整"的循环，才能最终找到适合自己的市场切入口。

## 市场分析和竞品研究的时间规划

最后我要讲一个实际的问题：这些工作需要花多长时间？

如果你已经有行业经验，可能1-2周就能完成。如果你是新手，建议花3-4周的时间。这个时间投入是值得的，因为：

- 第1周：收集数据阶段。下载报告、列竞品、收集基础信息。这一周会感到信息繁多、有些混乱，这是正常的。
- 第2周：整理和分析阶段。制作汇总表格、对比数据、找规律。这一周开始有感觉了。
- 第3周：深度思考阶段。反复看数据，问"为什么"，形成自己的观点。这一周开始有真正的认知。
- 第4周（可选）：补充验证阶段。对不确定的点做额外调研或实际体验，最后锁定你的策略。

这个过程不要加速，不要跳跃。很多创业者因为急着开始项目，跳过了第3周的深度思考，结果后面走了很多弯路。建议你把这几周的投入当作"项目启动的必要成本"，而不是"浪费时间"。
