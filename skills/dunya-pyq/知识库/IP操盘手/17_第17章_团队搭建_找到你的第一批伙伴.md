# 第17章 团队搭建：找到你的第一批伙伴

有个做小红书的女博主，叫琳琳。她做内容做了两年，粉丝从0做到了50万，每条笔记平均5000个赞。转化的事情也还不错，一年能出100万的销售额。

但从第三年开始，她发现了一个问题：选题、写文案、制作素材、拍照、回复私信、处理订单、跟进售后，所有的事都是她一个人干。她每天从早上六点忙到晚上十点，周末也不敢休息。焦虑的是，越来越多的选题想法浮现在脑子里，但时间根本不够用。一个月只能发十条内容，而她知道，做到年50万的销售额，流量已经开始天花板了。

有一天，她在镜子里看到自己，眼袋特别深。她突然意识到一个残酷的事实：**她不是在做一个IP生意，她是在做一份高薪工作。而且这份工作没有老板给她分配，没有同事帮她分担，所有的成败都压在她一个人身上。**

她做了一个决定：招人。

三个月后，她组建了一个三人团队——一个内容编导、一个运营负责人、一个客服。再后来的两年里，她的销售额从100万增长到了800万，但她每天工作的时间反而减少到了四小时。

这就是个人IP走向成熟的转折点。从"个人品牌"升级到"工作室"，从一个人的生意升级到一个组织的生意。这一步，决定了你的IP能走多远。

很多人对这一步有误解。他们觉得"等我一个人忙不过来了，再招人"。这个想法错得离谱。正确的做法是：**未来的你需要一支什么样的团队，今天就应该开始为那个团队做准备。**

这一章，我们来讲怎么找到第一批伙伴，怎么让他们为你的IP狂奔，以及怎么确保这个组织能像活的生物一样生长。

## 17.1 人才评估：冰山模型与ROI导向招聘

一个很多创业者容易掉的坑是：**急功近利地招人。** 当工作量堆到头顶的时候，他们会降低招聘标准。"只要会做内容就行"、"只要能吃苦就行"。然后三个月后就后悔了，因为招进来的人不仅没有减轻负担，反而成了新的负担。

为什么会这样？因为他们只看到了冰山之上的技能，而忽视了冰山之下的素质。

想象一下冰山这个比喻。浮出水面的部分，是可见的技能——这个人会视频剪辑，那个人懂社群运营，另一个人会数据分析。但水面之下，还有更大、更关键、也更难被看到的部分——学习速度、认知Ego、沟通能力、抗压能力、内在驱动力。

一个只有冰山之上的人，可能看起来很专业，但用不了多久就会显示出局限性。比如你招了一个"专业视频剪辑师"，他前公司用Adobe套件，你这儿用剪映，他说"我没用过这个软件，不会"。或者他已经形成了固定的工作方式，你建议他换个思路，他的反应是"我以前的公司都是这么干的"。这种人看起来技能很硬，实际上是个沉重的包袱。

而一个冰山之下扎实的人，哪怕目前的技能只有60分，但他的学习速度快、Ego低、愿意沟通。三个月后他可能已经是90分了。而且他会主动思考怎么做得更好，会跟你一起升级工作流程。

所以，**选人的黄金法则是：宁要70分基础+90分潜力的人，也不要100分技术+30分学习能力的人。**

这个判断标准，本质上就是一个ROI思维——**投入产出比**。不是选最便宜的人，也不是选最贵的人，而是选投入产出比最高的人。

怎么算这个ROI呢？有个简单的公式：

```
人才的价值 = 冰山之上的技能 × 冰山之下的潜力 × 价值观匹配度
月度成本 = 基本工资 + 管理成本 + 机会成本

ROI = 人才的价值 / 月度成本
```

这个"人才的价值"是个乘法，不是加法。意思是说，如果任何一个维度是0，整个价值就是0。一个技术特别强但Ego特别大的人，他的价值就是0。一个人品很好但学习能力很差的人，在快速变化的环境里，他的价值也接近0。

现在再回头看琳琳的例子。她招的三个人，表面上看起来都是"能力差不多"的，但实际上：

内容编导是从一个日更十条的小红书博主里挖出来的。这个人的技能基础只有70分，但她在快节奏环境里被训练过，学习速度特别快，而且特别能吃苦。她加入琳琳的团队后，第一周就提出了三个优化选题的想法。

运营负责人是琳琳的朋友推荐的。这个人从来没做过小红书，但他在前公司管理过社群，理解用户的心理。他上手特别快，一个月就建立了完整的社群运营系统。

客服是找的一个即将大学毕业的女生。说实话，她的"客服技能"是零。但琳琳喜欢她的一点是，面试时问她"怎么处理一个退货的用户？"，她没有背标准话术，而是真诚地说"我会先好好听他的意见，努力理解他为什么不满意"。这个姿态，比什么话术都值钱。

从技能角度，这三个人都不是"最专业的"。但从ROI角度，他们是"最合适的"。

那么在一个新成立的IP团队里，哪些岗位是必配的呢？

根据不同的IP类型，必配的核心岗位有三个：**内容、运营、商务**。

**内容岗**负责选题、策划、文案、素材这一套。如果你是视频型IP，可能还需要剪辑。如果你是长文型IP，可能需要深度写手。这个岗位的核心素质是：快速学习能力、内容敏感度、执行力。薪资范围，一线城市8k-15k，二线城市5k-10k。如果是兼职形式，按件计费或者提成制，可能1-3k一个月。

**运营岗**负责粉丝互动、社群维护、数据复盘、选题论证这一套。这个岗位的核心素质是：细致度、同理心、持续学习。有的IP会把运营和商务合并。薪资范围，一线城市6k-12k，二线城市4k-8k。

**商务岗**负责广告接洽、品牌合作、用户转化、售后这一套。这个岗位最考察人的"博弈能力"——既要维护用户体验，也要挖掘商业价值。薪资范围，一线城市10k-20k，二线城市6k-12k。如果是提成制，可能是基础4k+提成。

很多IP在初期会采用**兼职模式**而不是全职。这有两个好处：第一，成本更低。第二，风险更小——你可以在兼职的过程中观察这个人是否真的靠谱，再决定是否转成全职。我建议的做法是，前三个月都用兼职，看这个人的执行质量、沟通协作、契合度，然后再做全职决定。

兼职和全职的选择，本质上还是在问一个问题：**这个岗位对IP的增长是否关键？**

比如，内容岗对琳琳的IP来说是关键岗位，因为内容质量直接决定粉丝增长和粉丝质量。所以一开始就应该招全职。运营岗相对不那么关键——你可以自己管理社群，也可以招兼职。商务岗在早期也可以兼职，因为广告接洽的量级可能还不大。

但随着IP的成长，"关键岗位"的定义会变化。当运营的工作量从每天2小时变成8小时的时候，就该从兼职升为全职了。

最后再强调一遍核心的招聘哲学：**一个人的现有技能很容易被高估，他的学习能力和性格很容易被低估。但在快速变化的环境里，后者往往比前者更值钱。**

一个皮实耐操、Ego低、学习快的人，就像一块好的陶土，你可以捏成任何形状。而一个技能特别强但非常固执的人，就像一块生铁，再厉害的工匠也拿他没办法。

我见过太多这样的例子。有个内容IP的创始人，招了一个"前字节跳动的资深编辑"。简历特别漂亮，但入职一周就开始挑战创始人的选题——"这个选题不适合推荐流，我建议改成搜索向"。创始人解释自己的逻辑，他却说"但我在大厂就是这么做的，效果特别好"。一个月后，团队的士气就被搞得很低落。最后还是以不欢而散告终。

反而是另一个IP，招了一个刚毕业的"小白"，不会剪辑、不会运营、什么都不会。但这个小白有个特点——特别愿意问问题，特别愿意听建议，特别愿意加班学习。六个月后，这个人已经成了团队里不可或缺的中流砥柱。而且因为他是"自己带大的"，特别认同团队的文化，工作特别有劲头。

所以，下次招人的时候，不要过分迷信"大厂背景"、"专业资质"这些显眼的标签。反而要多看看这个人有没有"学习的渴望"、有没有"团队合作的意愿"、有没有"自我反思的能力"。这些东西看不见，但价值无穷。

还有一点容易被忽视：**警惕那些"太完美"的候选人。** 一个简历无懈可击、面试表现堪称教科书的人，往往是个"面试高手"。他能在面试里说出你最想听的话，但实际工作起来，可能对不上。

真正优秀的候选人，往往在面试里会透露出一些"不完美"——比如，他会坦诚地说出自己的不足，会问一些深入的、甚至有点"扎心"的问题，会在某些方面表现出真诚的困惑而不是标准答案。这恰好说明，他是一个"真人"而不是"演员"。

## 17.2 面试实战：五个维度的人才评估

招聘的时候，该问什么问题？

很多人的面试方式是：递上简历，问几个标准问题，然后决定要不要。"你的优点是什么？""你为什么离开上一家公司？""你对这个岗位怎么理解？" 候选人早就背好了标准答案，两分钟内把面试官唬得一愣一愣的。然后三个月后，你发现被唬了。

根本原因是，那些标准问题问不出真相。它们都指向候选人现在是什么，而不是候选人未来能成为什么。

正确的面试方式，是通过**五个维度来考古**——不是考察他们能说什么，而是考察他们曾经做过什么。人是他们过往经历的总和，行为模式有极高的路径依赖。

**第一维度：专业能力与业务理解**

最高明的问法不是"你懂小红书运营吗？"，而是"请用你自己的话，给我讲讲小红书平台的底层逻辑。包括它的推荐机制、什么样的内容容易被推、为什么。"

通过他的复述，你能看到三样东西：
- 他的倾听能力。他能不能准确理解你说的东西，还是开始跑题？
- 他的系统思维。他能不能把零散的知识点串成一个逻辑体系，还是只会背几个名词？
- 他对这个行业的真实理解。是真的理解，还是纸上谈兵？

然后问第二个问题："听了我的描述，你觉得这个逻辑有什么问题或者可以优化的地方吗？"这能看出他有没有独立思考的习惯。

**第二维度：内在驱动力与社会阅历**

这是最容易被忽视但也最重要的维度。一个人的工作积极性，往往来自于某个原始的、深层的内在动力。

直接问"你为什么想做这份工作？"通常得到的是虚伪的答案。真正能问出来的问法是："你为什么离开家乡来到这个城市？"、"你的原生家庭背景是什么样的？"、"你经历过最困难的时刻是什么？"

这些问题看起来跟工作没关系，但它们能透露出一个人最真实的驱动力。

一个从小镇来到大城市奋斗的人，往往有一种"要向父母证明自己"的原始动力。这种动力是最持久的——他会自驱地工作，不需要你督促。而一个生活无忧、没有外部压力的人，工作就会显得有点"应付"。

还要问："你最近一段时间最大的挑战是什么？你怎么克服的？"这能看出他的抗压能力是不是真的。有的人说"我特别能吃苦"，但当你问细节的时候，他描述的"加班到十点"，对于要做IP的人来说其实算不上什么。而有的人虽然没有刻意强调吃苦，但他讲述自己如何在绝境中找到办法的故事，就特别有说服力。

**第三维度：认知高度与学习能力**

问他："最近你在关注哪些内容创作者或博主？为什么关注他们？"

如果他说"我最喜欢某某博主，因为他说话拽拽的，特别有范儿"，说明他的认知可能还停留在表层——被人格魅力吸引，没有深入思考。

如果他说"我最近在看某某博主，他的框架思维特别强，每期节目的三点启发都能串成一个完整的知识体系。这给了我启发，我在运营社群的时候也开始尝试这种结构化的表达"，说明他不仅有高度的认知，而且有迁移能力——把别人的方法创意地用在自己的工作里。

然后问："上周有什么新学到的东西吗？"这很直白，但很有效。如果一个人说"没想起来"或者说"最近特别忙，没时间学"，说明他的心智已经关闭了。在快速变化的环境里，心智关闭的人就是定时炸弹。

**第四维度：沟通协作与抗压能力**

问他："你在上一份工作中，跟领导或同事发生过矛盾吗？怎么处理的？"

这个问题有陷阱的。如果他说"没有，我从不与人产生矛盾"，那他要么是在撒谎，要么就是特别老实但也特别"无趣"。

真实的人会说"有过矛盾，当时我和运营负责人对某个内容的方向有分歧"。然后你追问："后来怎么处理的？"

好的答案是："我没有坚持自己的想法，而是先认真听他的逻辑。后来才意识到，他的角度其实是对的，我有些想法太主观了。"这说明他有合作精神、有自我反思的能力。

不好的答案是："我最后坚持了我的想法，事实证明我是对的。"这说明Ego特别强，很难合作。

还要问："你能接受加班吗？"这个问题要看微表情。身体前倾、眼神发光的人，说的"能接受加班"是真的。身体后仰、眼神游离的人，说的就是虚话。

**第五维度：长期主义与价值观匹配**

问他："你三到五年的收入目标是多少？"这能看出他对现实的感知。如果一个做内容编导的人说"三年后我要月薪50万"，要么他对市场的认知有问题，要么就是在说假话。而如果他说"三年后月薪15到20万，然后考虑自己创业"，说明他有真实的、可达成的目标。

然后问："赚到钱之后，你想做什么？"

能接受"赚钱后我想回家陪陪父母"、"我想用这笔钱支持我的爱好"的人，说明价值观相对平衡。而如果回答是"赚到钱后我就辞职，再也不工作了"，说明他对工作的理解就是纯粹的生存需求，长期动力有限。

最后，问他对你的IP有什么看法。真正感兴趣的人会说出具体的观察："我看过你十几条视频，我发现你的选题特别接近你粉丝的痛点，但我觉得可以在……方面做得更好。"而敷衍的人就是"你的内容特别棒"这样的万能评价。

整场面试下来，你其实不是在评估他"会不会做这份工作"，而是在评估他"是不是咱们这队的"。因为一个合适的人，他会自己学会技能。而一个不合适的人，再怎么教也教不出来。

## 17.3 招聘节奏：常态化而非应急式

很多IP创始人犯的一个错误是：**等到忙不过来了，才开始招人。**

你能想象一个足球队，等到球员累倒了，教练才开始替补吗？这简直是自杀性的策略。

正确的做法是：**根据你的增长曲线，提前6到12个月开始为未来的团队做准备。**

换句话说，你需要对自己做一个诊断：如果按照目前的增长速度，一年后我需要什么样的团队？那我从现在开始就应该去寻找这样的人。

这就是**常态化招聘**的逻辑。它意味着：

**第一，把招聘变成一个固定的、定期的动作，而不是应急反应。**

我建议每周固定拿出6小时用于面试和招聘。不是"我有空就面试"，而是"每周三晚上7点到9点，还有周六下午3点到5点"。这样做的好处是，你能持续了解市场上都有什么样的人才，他们的行情怎么样，什么样的人能打动你。而且，当真的需要一个人的时候，你已经见过一轮候选人了，反应速度会快很多。

**第二，建立一个人才储备池。**

每次面试，即使这次没有合适的岗位，但如果这个人的素质还不错，就应该留下他的信息。"我现在没有岗位给你，但我特别看好你。三个月后我可能需要一个运营负责人，到时候能不能找你？" 很多人听到这个，会很感动。而且你保持定期的沟通——每个月发一条信息问候一下，分享一些行业见解，邀请他参加你的线下活动。这样，当你真的需要人的时候，他的第一反应就是"我想去"。

**第三，多渠道招聘。**

常见的招聘渠道有：

- **朋友推荐**：最有效，也最容易快速评估。你信任你的朋友，你朋友推荐的人，背书就在那儿。
- **行业社群**：参加一些行业的微信群、线下沙龙，多交流。很多优秀的人才就在这些地方活跃。
- **招聘平台**：拉勾、BOSS直聘、脉脉等。虽然转化率不如前两个，但能看到大量候选人，概率还是有的。
- **内容呐喊**：在你的平台上，偶尔提一下"我在招什么样的人"。你的粉丝里可能就有很多潜在的优秀团队成员。
- **大学校招**：如果你要找的是实习生或应届毕业生，可以去大学里校招。成本低，而且这些人可以在你这儿获得很好的培养。

我特别推荐**朋友推荐**这个渠道。因为最好的人才往往不在招聘市场上，他们已经在别的地方工作了。而让他们愿意离开的唯一方式，就是一个他信任的人的推荐。

**第四，快速淘汰。**

但这里有个很容易被忽视的细节：你要敢于快速拒绝和淘汰。

拒绝候选人的时候要快，不要让人家等两周。淘汰不匹配的员工的时候也要快，不要等到他带坏了整个团队的气氛。

琳琳曾经招过一个内容编导，表面上能力不错，但入职后发现他特别"事儿"——要求这个改这个，那个改那个，总觉得自己的想法最对。一开始琳琳想"再观察一下"，结果两个月的时间被白白浪费了，而且这个人开始影响其他员工的工作态度。最后还是得开除，但这已经造成了很多损伤。

如果及早决定"这个人不匹配"，可能第一个月就会做出这个决定，损失就小得多。所以，**对不匹配的人，要有"快刀斩乱麻"的魄力。**

**第五，团队文化的提前准备。**

招人的时候，很多人只想到了"能力搭配"——我需要一个内容手，一个运营手，一个商务手。但忽视了一个更重要的问题：**这些人能不能合在一起工作？他们的价值观一不一致？做事的节奏一不一致？**

这些问题决定了团队能走多远。一个能力互补但三观不合的团队，迟早会分裂。而一个能力稍弱但价值观统一、做事节奏一致的团队，反而能爆发出巨大的力量。

所以，在招人的时候，要特别问一个问题："你怎么理解IP这个事？你觉得什么样的IP值得为之奋斗？" 通过这个问题，你能看出候选人对"这个事业"的理解是不是跟你一致。

有的人觉得IP就是"快速圈钱"，有的人觉得IP是"建立长期的信任"。这两种人放在一起工作，会经常产生冲突。而如果他们的IP观念从一开始就一致，后续的合作就会顺很多。

最后一个建议：**不要追求大而全的一步到位。而要追求快速迭代。**

很多创始人想"一次性把完整的团队搭好"——内容、运营、商务、行政、财务，一步到位。结果是成本太高，而且很多岗位的人力还没有充分利用，就变成了"为了有个完整的组织而组织"。

更聪明的做法是，先搭最必要的岗位，然后根据增长情况逐步补充。冷启动期，可能就是内容+兼职运营。成长期，再加上商务。成熟期，再加上行政和财务。这样做，既能控制成本，也能保持团队的精悍。

## 本章小结

**第一个关键认知：人才评估的黄金法则是"冰山模型"。** 不要被表面的技能迷惑，要看透水面之下的学习能力、认知高度、内在驱动力。一个70分基础+90分潜力的人，远比100分技术+30分学习能力的人更值钱。

**第二个关键认知：面试的本质是考古，不是考试。** 通过五个维度——专业理解、驱动力、认知高度、协作能力、价值观——来透视一个人的真实素质。不要被他们怎么说迷惑，要看他们曾经怎么做。

**第三个关键认知：招聘应该常态化，而不是应急。** 每周固定时间面试，建立人才储备池，多渠道招聘，快速淘汰不匹配的人。这样当真正需要人的时候，你已经有充足的准备了。

**第四个关键认知：组织的成功50%来自选人。** 能力可以培养，但价值观、驱动力、协作精神很难改变。所以选人的时候，要把"这个人是不是咱们队的"作为第一考量。
