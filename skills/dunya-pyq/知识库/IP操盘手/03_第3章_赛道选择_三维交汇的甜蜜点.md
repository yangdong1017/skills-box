# 第3章 赛道选择：三维交汇的甜蜜点

我的一个朋友老陈，做了二十年律师，专攻劳动争议。他决定做IP的时候，问我："我做什么赛道好？"

我反问他："你觉得呢？"

他说："我想做法律科普。抖音上法律类账号很火，而且我有专业能力。"

我又问："法律科普这个赛道有多少人在做了？"

他说不知道。我让他搜了一下"律师"相关账号，发现头部法律科普账号已经有好几个，粉丝动辄几百万，而且风格各异——有犀利型的、有温暖型的、有搞笑型的。

"那你的差异化在哪里？"

他想了想说："我比他们更专业？"

但问题是，观众分不出"更专业"和"一般专业"的区别。当你说"我的法律知识更扎实"的时候，观众看到的只是又一个穿西装讲法条的律师。

后来我帮他做了一个分析。他的真正优势不是"法律知识"——这个很多人都有——而是"二十年劳动争议的实战经验"。他处理过的奇葩案例能讲三天三夜。所以他最终的定位不是"法律科普"，而是"打工人维权故事"。

赛道一换，粉丝画像一下子清晰了——不是所有对法律感兴趣的人，而是那些觉得自己在职场被不公正对待的人。内容形式也清晰了——不是干巴巴地讲法条，而是讲真实的维权故事。变现路径也清晰了——不是卖"法律入门课"，而是承接劳动仲裁的咨询和代理。

这就是赛道选择的力量。选对了赛道，后面的每一步都顺；选错了赛道，后面的每一步都拧巴。

---

## 3.1 赛道评估的三大核心指标

很多人选赛道靠的是"感觉"——觉得某个领域火就去做。但感觉往往不靠谱。一个看起来很火的赛道，可能已经挤满了人；一个看起来冷门的赛道，可能藏着巨大的蓝海机会。

评估赛道需要看三个硬指标。

**第一个指标：需求密度。**

这个赛道有多少人在关注这个话题？搜索量、话题阅读量、相关内容的播放量——这些数据可以告诉你需求有多大。

判断需求密度的方法很简单：去各平台搜你打算做的关键词，看搜索联想词有多少、相关内容的平均播放量有多高、相关话题的累计阅读量有多大。如果一个关键词搜出来只有寥寥几条内容，说明需求密度太低；如果搜出来铺天盖地都是内容，说明需求密度够高——但同时也意味着竞争可能很激烈。

**第二个指标：供给饱和度。**

这个赛道已经有多少创作者在做了？头部账号有几个？腰部账号密度如何？新账号还有没有成长空间？

供给饱和度和需求密度要对着看。理想的赛道是"高需求+低饱和"——需要这类内容的人很多，但做这类内容的人还不多。这就是通常说的蓝海。

不过，纯蓝海几乎不存在了。更现实的情况是找到一个大赛道里的小蓝海。比如"健身"是红海，但"40岁以上男性的居家力量训练"可能就是蓝海。"育儿"是红海，但"二胎家庭的老大心理调适"可能就是蓝海。

**第三个指标：传播效率。**

这个赛道的内容，用户愿不愿意分享？

有些赛道天然有高传播性——搞笑、情感共鸣、社会话题，用户看了就想转发。有些赛道天然传播性低——比如专业工具教程，用户觉得有用但不会分享给朋友，因为"太垂直了，分享出去别人也不感兴趣"。

传播效率高的赛道，你的增长会更快，因为每一条内容都有可能被用户主动传播。传播效率低的赛道不是不能做，但你需要更依赖搜索流量和平台推荐，而不是用户的社交传播。

```
赛道评估矩阵

         需求密度 高
              ▲
              │
    蓝海机会   │    红海竞争
    高需求低供给│    高需求高供给
    最佳入口点  │    需强差异化
              │
──────────────┼──────────────▶ 供给饱和度 高
              │
    冷门利基   │    死亡地带
    低需求低供给│    低需求高供给
    需深耕     │    避免进入
              │
         需求密度 低
```

最理想的位置是左上角——蓝海机会区。但如果你发现自己感兴趣的领域在右上角（红海），也不要绝望。关键是进入下一步：找到差异化的切入点。

## 3.2 定位罗盘：能力、需求、竞争的三维交汇

评估完赛道的大环境，下一步是找到你在这个赛道里的具体位置。这需要把三个维度对齐。

**第一个维度：市场需求。** 你的目标用户有哪些未被满足的痛点？他们在搜什么、问什么、抱怨什么？这些痛点中，哪些是高频的、强烈的、愿意付费解决的？

**第二个维度：个人禀赋。** 你有什么独特的能力、经验或资源？注意不是"你觉得自己哪里好"，而是上一章说的"真有资产"——经过验证的、有沉没成本的真实优势。

**第三个维度：竞争格局。** 现有的竞争者是谁？他们的定位是什么？他们的弱点在哪里？有没有他们没有覆盖到的空白区域？

你的最佳定位，就是这三个维度的交汇点——市场有需求，你有能力满足，竞争对手又没有覆盖到的那个点。

```
              市场需求
                △
               / \
              /   \
             / ★   \        ★ = 你的甜蜜点
            /  你的  \
           / 最佳定位 \
          /           \
         ▽─────────────▽
    个人禀赋          竞争格局
```

这三个维度缺一不可。只看需求不看能力，你会冲进一个自己做不好的领域——比如你文字功底很好但镜头表现力差，非要去做口播型内容。只看能力不看需求，你会做出"自嗨型"内容——你确实很懂，但没人在乎。只看需求和能力不看竞争，你可能冲进一个已经被头部占满的赛道——你做得再好也很难出头。

具体操作可以分三步。

**第一步：需求审计。** 打开各大平台，搜索你计划进入的领域。看用户在评论区问什么问题、在搜索框搜什么关键词、在问答社区提什么问题。把这些需求整理出来，按照"频率×痛苦程度×付费意愿"排序。排在前面的，就是你最应该优先覆盖的需求。

**第二步：资源审计。** 上一章做的资产审计在这里派上用场了。你的"真有资产"列表上，哪些资产可以转化为满足上述需求的能力？比如你有十年HR经验（真资产），用户有"不知道怎么写简历"的高频痛点（强需求），那么你就有能力做"简历优化"这个方向。

**第三步：竞争扫描。** 看看目前在做类似内容的账号有哪些。他们做得好的地方是什么？做得不够好的地方是什么？有没有他们没有覆盖到的角度或人群？

把这三步的结果叠在一起看，你通常就能找到那个"甜蜜点"——一个需求真实存在、你有能力满足、竞争相对空白的位置。

还是老陈的例子。法律科普的需求是真实的（需求审计通过），他有专业能力（资源审计通过），但竞争太激烈（竞争扫描不通过）。缩小到"劳动争议维权"之后，需求依然真实（打工人维权是高频痛点），他的能力完全匹配（二十年劳动争议经验），竞争大幅减小（专做劳动争议故事的账号很少）。三维交汇，甜蜜点找到了。

## 3.3 差异化的四种切入方式

即使找到了赛道里的大方向，你仍然需要差异化。因为用户关注你的理由不是"你也做这个领域"，而是"你做这个领域的方式跟别人不一样"。

差异化有四种常见的切入方式。

**第一种：品类切割。**

把一个大品类切成更小的细分品类，然后占领那个小品类。

"做菜"是一个大品类，竞争极度激烈。但"一人食快手菜"就是一个切割后的小品类——专门针对独居年轻人，做那种15分钟以内、一个锅搞定的菜。大品类里你排不上号，小品类里你可以做到第一。

切割的方法是加限定词。"健身"太大，加上"居家"变成"居家健身"，再加上"30岁以上女性"变成"30岁以上女性居家健身"——越精确，竞争越小，用户越觉得"这就是为我做的"。

当然，切得太细也有风险——受众可能太小了，天花板太低。所以品类切割需要在"精准"和"规模"之间找到平衡。一个可操作的经验法则是：你切出来的细分品类，在平台上搜索的时候应该有一定的内容量（说明有需求），但头部账号不超过3到5个（说明供给不饱和）。

**第二种：元素重组。**

把两个看似不相关的领域嫁接在一起，创造出新的内容形式。

"法律+脱口秀"——用脱口秀的形式讲法律知识，罗翔老师就是这条路线的标杆。"健身+科学"——不是教你怎么练，而是用科学论文告诉你为什么要这样练。"理财+动画"——用动画短片的形式讲理财概念。

元素重组的关键是两个元素之间有"认知差"——用户在A领域看到B领域的东西，会产生新鲜感和好奇心。比如用户在健身领域看到学术论文式的内容，会觉得"这个人好专业"，这个新鲜感本身就是差异化。

但重组不是随便搭配。两个元素之间必须有逻辑上的关联——"法律+脱口秀"成立，因为法律故事本身就有戏剧性；"法律+美食"就不太成立，因为两者之间缺乏自然的连接。

**第三种：反向定位。**

当赛道里所有人都在往一个方向走的时候，你往反方向走。

比如，所有的理财账号都在教你怎么赚钱，你做一个教你"怎么少亏钱"的账号。所有的美妆博主都在推新品，你做一个"什么不值得买"的测评号。所有的育儿账号都在讲"如何做一个好妈妈"，你做一个"不完美妈妈也很好"的情感账号。

反向定位能迅速抓住注意力，因为它制造了认知冲突——用户刷到一个跟其他内容调性完全不同的东西，会本能地停下来看。但反向定位有风险：如果你的"反向"观点经不起推敲，用户会觉得你是在哗众取宠。所以反向定位的内容需要更强的论证能力。

**第四种：人格符号化。**

用一个极度鲜明的个人特征作为差异化标志。

可以是一个视觉符号——比如永远戴一顶红帽子、永远穿白大褂。可以是一个语言习惯——比如每条视频结尾都用同一句口头禅。可以是一个行为特征——比如总是在街头随机采访路人。

人格符号的作用是降低用户的记忆成本。在海量内容中，用户记不住你的名字，但可能记住"那个戴红帽子的健身博主"。一旦这个符号被记住了，你在用户心智中就有了一个独特的位置。

```
四种差异化切入方式

方式         核心逻辑              适用情境              风险点
───────── ─────────────────── ────────────────── ──────────────
品类切割    大变小，占领细分       赛道拥挤，需缩小战场   切太细天花板低
元素重组    A+B，创造新鲜感       有跨领域经验或视角     两元素缺乏关联
反向定位    众人向左，你向右       赛道同质化严重         论点经不起推敲
人格符号    一个标志性特征         个人特质突出           符号太刻意适得其反
```

四种方式不是互斥的，可以组合使用。比如"品类切割+人格符号"——在一个细分品类里用一个鲜明的个人特征做标识。

## 3.4 对标分析：三个可复制的维度

找到了差异化方向之后，下一步是做对标分析——找到你这个赛道里做得好的人，研究他们的成功要素，取其精华为己所用。

注意：对标不是模仿。你不是要变成另一个人，你是要从别人的成功里提取"可迁移的规律"，然后用你自己的方式实现。

对标分析可以从三个维度入手。

**维度一：内容结构。**

头部账号的内容是怎么组织的？他们的视频开头通常用什么方式抓注意力——是提问、是悬念、是冲突？中间怎么推进——是故事线、是步骤式、是对比式？结尾怎么收——是金句总结、是行动号召、是下期预告？

内容结构是最容易复制的东西。你不需要抄他的选题和文案，但完全可以学习他的结构框架——比如"先抛问题→讲一个反面案例→给出正确方法→总结"这种结构，适用于几乎所有知识类内容。

**维度二：战术技巧。**

他们用了哪些具体的技巧来提升数据？比如，评论区是怎么互动的？标题是怎么写的？是不是在视频里设计了引导关注的环节？有没有固定的发布频率和时间？

战术技巧是有时效性的——今天管用的技巧明天可能过时。但其中有一些是相对稳定的，比如"在视频前三秒设置钩子""在评论区置顶一条引导互动的评论"——这些是基础操作，值得学习。

**维度三：商业模型。**

这是最重要也最容易被忽略的维度。头部账号是怎么赚钱的？他们的产品是什么？产品线是怎么搭建的？从哪个产品切入、后面怎么延伸？价格怎么定的？转化路径是怎样的——从内容到加微信到成交，每一步是怎么设计的？

商业模型的对标分析能帮你避免一个常见错误：做了半年内容，发现不知道怎么变现。如果你提前研究了对标账号的商业模型，你就能在一开始就把变现路径设计好。

做对标分析有一个很实用的方法：**选3到5个对标账号，从三个维度分别打分，然后做一张对比表。**

| 维度 | 对标A | 对标B | 对标C | 我的计划 |
|------|-------|-------|-------|---------|
| 内容结构 | 故事+干货 | 纯口播 | 场景化演示 | ？ |
| 核心技巧 | 评论区互动强 | 标题震撼 | 视觉精美 | ？ |
| 商业模型 | 卖课+咨询 | 接广告 | 电商带货 | ？ |
| 值得学习的 | 叙事方式 | 选题角度 | 视觉风格 | 整合到自己的系统 |
| 可以超越的 | 更新太慢 | 内容太浅 | 没有互动 | 作为我的差异化 |

最后一行尤其重要——"可以超越的"就是你的机会点。如果对标A做得很好但更新频率低，而你能做到高频率输出同等质量的内容，那"稳定的高频更新"就是你超越他的切入点。

记住：对标分析的目的不是"变成他"，而是"变成更好的你"。你从对标A那里学到了叙事方式，从对标B那里学到了选题思路，从对标C那里学到了变现路径，然后用你自己的人格和方式把它们整合在一起。这个整合后的版本，就是独一无二的你。

## 3.5 从「专家」到「翻译官」：找到你的表达位置

赛道选好了，差异化找到了，对标也做了。最后还有一个至关重要的问题：**你在用户面前扮演什么角色？**

很多专业人士做IP的时候，有一个本能的反应——"我是专家，我来教你"。律师讲法律、医生讲健康、财务讲税务——端着专家的架子，用专业术语，自上而下地"教"。

这是一个陷阱。叫做**"专家的诅咒"**。

什么是专家的诅咒？就是你在一个领域待得太久了，已经无法想象一个完全不懂这个领域的人是什么感受。你觉得理所当然的概念，对普通人来说完全是天书。你说"仲裁申请书需要明确诉讼请求和事实理由"，用户听到的是"一堆我听不懂的词"。你说"CPM和CPC的区别是……"，用户的反应是"能说人话吗？"

更糟糕的是，专家姿态会制造距离感。当你站在讲台上"教"的时候，用户的心理位置是"台下被教育的人"。这种不对等的关系，在传统课堂里可以接受，但在社交媒体上会让人不舒服。因为用户打开抖音是来找乐子的，不是来上课的。

更好的角色定位是**"翻译官"**或者**"启蒙者"**。

翻译官做的事情是：把专业世界的语言，翻译成普通人听得懂的话。你不是在"教"用户，你是在"帮"用户——帮他理解一个他原本觉得很复杂的东西。

启蒙者做的事情是：带用户看到他此前没有注意到的东西。不是居高临下地说"你不懂"，而是站在用户身边说"你看，这里有一个有趣的东西，我带你看看"。

具体到内容创作中，这意味着三个转变：

**第一，少用术语，多用比喻。** 不要说"核心竞争力的构建需要差异化的价值主张"，要说"在一条街上开第十家奶茶店，你凭什么让人进你的店而不是别人的？"

**第二，少说教，多讲故事。** 不要说"你应该这样做"，而是"我认识一个人，他遇到了跟你一样的问题，然后他做了这件事，结果怎样怎样"。故事比道理有说服力得多。

**第三，少高冷，多共情。** 不要站在"我都会"的位置上俯视用户，而是回忆一下自己当初"什么都不会"的时候是什么感受。用这种感受来与用户建立情感连接。

```
专家姿态                         翻译官/启蒙者姿态
────────────────────              ────────────────────
关系：我教你                      关系：我带你看
语言：专业术语                     语言：生活化比喻
态度：我都会，你不会               态度：我也是从不会走过来的
内容目标：展示我的专业              内容目标：解决你的问题
用户感受：被教育、有距离            用户感受：被理解、有共鸣
```

最后说一个关于"表达位置"的深层选择：**你想成为网红还是标准？**

网红型IP靠的是个人魅力——粉丝因为你这个人而关注你。这条路起步快，但也脆弱——如果你的人设翻车或者风格过时，粉丝会迅速流失。

标准型IP靠的是工具价值——用户把你当成某个领域的"参考手册"。比如在某个垂直领域，遇到问题第一反应就是"去搜XX的视频看看"。这条路起步慢，但更稳——因为你的价值不依赖于人格魅力，而是依赖于你持续输出的有用内容。

最理想的状态是两者兼具：有人格魅力（用户喜欢你），也有工具价值（用户需要你）。但如果你必须选一个先做到，建议先做工具价值。因为工具价值更可持续，也更容易变现——用户愿意为"解决问题"付费，但很少愿意为"喜欢一个人"付费。

---

## 本章小结

赛道选择不是拍脑袋的决定，它是一个需要数据支撑的理性分析过程。

**第一步，评估赛道大环境。** 看需求密度、供给饱和度、传播效率。找到蓝海机会或者红海中的空白区域。

**第二步，定位三维交汇。** 在市场需求、个人禀赋、竞争格局三个维度中找到交汇的甜蜜点。三个维度缺一不可。

**第三步，建立差异化。** 用品类切割、元素重组、反向定位或人格符号中的一种或几种方式，让自己在赛道中拥有独特的位置。

**第四步，做对标分析。** 从内容结构、战术技巧、商业模型三个维度分析对标账号，取其精华，找到超越机会。

**第五步，确定表达位置。** 做翻译官而不是做专家，用生活化的语言、故事化的表达、共情的态度来与用户建立连接。先做工具价值，再叠加人格魅力。

下一章，我们会在这个赛道的基础上，进一步打磨你的"人设"——从真实的你出发，锻造一个既真实又有商业价值的IP形象。
