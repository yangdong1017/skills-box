## 1. 抖音 | 短视频之王的流量密码

### 核心算法逻辑

**一句话总结：** 智能推荐+社交推荐，冷启动通过完播率→热度评分，热度高的内容跨兴趣标签推荐

### 最重要的3个指标

| 指标 | 目标值 | 含义 |
|-----|-------|------|
| 完播率 | ≥70%（15s内容），≥50%（30-60s内容） | 衡量内容吸引力的第一指标，直接影响推荐力度 |
| 点赞率 | ≥8%（优秀），≥5%（及格） | 用户对内容的认可度，决定是否继续推荐 |
| 评论率 | ≥2%（优秀），≥0.5%（及格） | 用户参与度，高于点赞率的内容往往容易出爆款 |

**核心公式：** 热度评分 = 完播率×30% + 点赞率×40% + 评论率×30%

### 内容偏好

什么类型容易获得推荐？

- ✓ **开头3秒吸引：** 第1秒就要让用户停下来；最有效的方式是悬念、反差、实用建议
- ✓ **垂直度高：** 账号越垂直，推荐越精准；跨界内容容易掉赞
- ✓ **高互动话题：** 文案结尾用疑问句引导评论，效果显著
- ✓ **时代感强：** 紧跟热点、新闻、梗的内容容易有初始流量
- ✓ **服装统一：** 同一人物的内容穿着、发型、背景保持一致，形成视觉记忆

### 发布时间建议

| 时间段 | 推荐指数 | 说明 |
|--------|--------|------|
| 6:00-8:00 | ⭐⭐⭐⭐ | 上班族通勤时间，打开率高 |
| 12:00-13:00 | ⭐⭐⭐⭐ | 午休黄金时间 |
| 18:00-20:00 | ⭐⭐⭐⭐⭐ | 晚间流量高峰 |
| 20:00-22:00 | ⭐⭐⭐⭐⭐ | 用户最活跃时段 |
| 22:00-24:00 | ⭐⭐⭐ | 次晚高峰 |

**建议：** 根据粉丝活跃时间分析，优先在目标用户最活跃的时段发布

### 新号冷启动建议

| 阶段 | 时间 | 重点工作 |
|-----|------|--------|
| 第1阶段 | 第1-7天 | 发布7条高质量内容，每条间隔≥12小时；不求数据，只求垂直度 |
| 第2阶段 | 第8-14天 | 分析这7条内容的数据；找到数据最好的1-2条的特征，复制这个特征 |
| 第3阶段 | 第15-30天 | 保持日更或周4以上的频率；继续优化完播率和互动率 |
| 第4阶段 | 第30天+ | 如果月均完播率>50%，账号基本定型，开始布局变现 |

### 常见误区

| 误区 | 正确做法 |
|-----|--------|
| 追求高端内容、大制作 | 专业度排在垂直度和吸引力之后；新号应该优先保证吸引力和完播率 |
| 一天发多条 | 算法会削弱后发的视频；一天最多发2条，间隔≥8小时 |
| 靠买粉丝或互赞 | 算法识别虚假互动，反而降权；务必拒绝任何刷粉互赞 |
| 频繁改账号简介 | 改动会重置标签，影响推荐精准度；账号定型后不要改 |
| 新号就考虑变现 | 新号第一目标是做到3000粉丝+5个视频≥100w播放量；才有变现价值 |

## 2. 小红书 | 社交电商的社区逻辑

### 核心算法逻辑

**一句话总结：** CES评分（完整度+互动度+传播度）决定推荐；社交推荐+信息流推荐双引擎

### CES评分体系详解

| 评分维度 | 权重 | 计算方式 | 优化方向 |
|---------|------|---------|--------|
| 完整度(C) | 30% | 内容图片数、文案长度、话题数、标签数 | 至少9张配图；文案200字以上；2-5个话题；3-5个标签 |
| 互动度(E) | 40% | 点赞数、评论数、收藏数（权重：评论>收藏>点赞） | 首小时内的评论数最关键；评论数×3倍的权重 |
| 传播度(S) | 30% | 分享数、转发数、话题参与数 | 鼓励评论分享；参与热门话题；激发转发欲 |

**核心公式：** CES = C×0.3 + E×0.4 + S×0.3

### 内容偏好

- ✓ **长文案：** 200-500字最优，文案长度越长CES评分越高
- ✓ **9张配图：** 图片数越多完整度越高；9张是小红书最优配置

- ✓ **有例子：** 用实际案例佐证观点；比纯理论更容易获赞

## 7. 微信公众号 | 私域流量的经营

### 核心算法逻辑

- ✓ **文章排版精美：** 公众号排版美观度影响阅读体验；要重视排版
- ✓ **定期更新：** 公众号更新的稳定性影响阅读量；建议周2-3更新

4. **Act（行动）：** 立即执行，比如更新一个文案、调整一个数据指标
