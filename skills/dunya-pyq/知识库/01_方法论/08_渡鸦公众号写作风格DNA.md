## 三、标题公式库

### 公式1：数字+结果承诺型
```
{数字}个{赛道}最新{技术}，{效果}（附{赠品}）
例：25年6月K12赛道最新截流话术30条
例：小红书5个技巧彻底引爆笔记互动与停留
```

### 公式2：反常识/颠覆认知型
```
{常见观点}纯属{否定词}，揭秘{真正原因}
例："养号"纯属伪命题，揭秘决定作品生死的四大核心要素
```

### 公式3：情绪标题党型
```
{情绪词}！{赛道/现象}，{夸张效果}
例：疯了！万亿赛道"算法漏洞"，普通人1秒躺赚
例：震惊！健康流量凉透？一键启动，独吞全网
```

### 公式4：人群定向型
```
{目标人群}必看，{效果}
例：医美老板/运营必看，提升10倍互动率的评论区截流密码
```

### 公式5：哲学/文学型
```
{金句/古语}，{副标题说明内容}
例：世人畏果我畏因，一个心法教你从本质上解决所有引流问题
```

### 公式6：品牌叙事型
```
写给{人群}的你，{情感描述}
例：写给2025还在牌桌上的你，惊涛骇浪之后我们如何颠覆式增长
```

## 四、开头Hook六种模式

### 模式1：痛点共鸣+自我带入（最常用）
```
兄弟们，你是不是也常常感觉：
{排比3个痛点场景}？
{情绪爆发句}！
{反转句}。
```

### 模式2：哲学/仪式感开场
```
{金句/哲学句}。今天这篇文章，不讲干货，也不卖课。
它很长，可能需要你泡杯茶，找个不被打扰的角落，慢慢看完。
```

### 模式3：场景还原+数据冲击
```
{具体场景描述}。
我问了一个问题："{互动问题}"
那一瞬间，屏幕上{群体反应}。
```

### 模式4：权威自我介绍
```
我是琛哥，渡鸦科技社的创始人。
{精确数据背书}。
```

### 模式5：焦虑/社会热点
```
{热点话题}，你是不是也{焦虑表现}？
{承认恐惧}，{承诺解决方案}。
```

### 模式6：终极金句
```
{对仗金句}。
{暗示价值}。
```

## 五、文章结构模板

### 模板A：品牌叙事型（英雄之旅）
```
1. 悬念/哲思开场（100-300字）
2. 繁荣巅峰（年会数据、高光时刻）
3. 危机降临（困境描述）
4. 艰难探索（笨办法、失败尝试）
5. 绝境转折（技术突破/灵感）
6. 翻盘展示（好评截图、数据）
7. 更大愿景（新产品/新方向）
8. 情感升华（"值了"）
9. 坦诚反思（"不会盲目自信了"）
10. 标准CTA收尾
```

### 模板B：干货教学型（道-术-法-器）
```
1. 痛点Hook
2. 琛哥自我介绍+数据背书
3. 核心方法论命名（自创术语）
4. 分层拆解 01/02/03...
   - 每层：定义 → 原因 → 方案 → 延伸
5. 实操演示/截图
6. 常见误区纠偏
7. 总结金句
8. 标准CTA收尾
```

### 模板C：产品推销型（价值堆叠）
```
1. 亲切问候+社交证明
2. 数据增长展示
3. "为什么要升级？"（时代背景）
4. 用户需求分析
5. 权益逐项展开（价值→外部价格→会员价）
6. "渡鸦的交付到不到位？"
7. 价格锚定（588/3年=一天不到6毛钱）
8. 反向筛选/劝退环节
9. 加入方式+FAQ
10. 情感升华+CTA
```

## 六、语气与用词规范

### 称呼体系
| 称呼 | 使用场景 | 频率 |
|------|----------|------|
| 兄弟们 | 默认称呼，几乎所有文章 | 70% |
| 兄弟姐妹们 | 正式场合（年终总结、产品发布） | 15% |
| 各位 | 更正式的演讲 | 10% |
| 姐妹们/老板们 | 特定赛道文章 | 5% |

### 语气三层切换
```
开头（口语+共鸣）→ 中段（导师腔/专业）→ 结尾（口语+热血）

口语端："嘎嘎出单！""卧槽这不就是我吗？""根本回不完"
专业端："核心逻辑是……""你需要理解的是……""本质上"
热血端："手握流量，兴风作浪！""渡鸦陪你，冲锋！"
```

### 禁止使用的词汇
- 敬语体："您好""请问""感谢您的关注"
- 学术腔："综上所述""研究表明""笔者以为"
- 广告腔："限时优惠""错过再等一年""赶快行动"
- 软弱表达："或许""可能""也许""尽量"

### 必须使用的词汇库
```
口语类：嘎嘎、牛逼、卧槽、YYDS、吃瓜、兄弟、靠谱
军事类：冲锋、阵地、弹药、攻城、运筹帷幄
武侠类：心法、绝学、内功、江湖、降维打击
情绪类：扎心、炸裂、绝望、翻盘、逆袭、兴风作浪
品牌类：渡鸦、琛哥、墨鸦AI、墨涅斯、青云、渡鸦科技社
```

## 十一、排版与格式规范

### 编号体系
- 使用 `01/02/03` 而非 `一/二/三`（互联网风格）
- 嵌套：01 > 1 > (1) > 核心/实战/延伸

### 段落节奏
- 每段不超过4行
- 关键信息单独成行
- 大量使用空行制造呼吸感
- 金句必须单独一行

### 强调方式
- 加粗：核心概念、关键数据、方法论名称
- 引号：术语标注（"万流归因诀""说人话"心法）
- 方括号：【渡鸦XX】品牌标注
- 括号补充：（吃瓜）（当然这只是一个例子）

### 截图/配图占位
```
[配图：{描述}]
（此处在实际发布时插入截图/图片）
```

## 十二、写作自检清单

每次写完文章后，对照以下清单检查：

- [ ] 开头是否用"兄弟们"+ 痛点共鸣建立连接？
- [ ] 是否包含琛哥自我介绍+精确数据背书？
- [ ] 是否有自创方法论/框架命名？
- [ ] 是否用 01/02/03 分层拆解？
- [ ] 语气是否在口语↔专业之间自然切换？
- [ ] 是否至少包含1个军事/武侠隐喻？
- [ ] 是否至少包含1句金句（可从金句库选用或自创）？
- [ ] 是否有用户反馈/数据作为社会证明？
- [ ] 结尾是否带标准CTA + "手握流量，兴风作浪"？
- [ ] 段落是否足够短（每段<4行）？
- [ ] 是否没有使用敬语体/学术腔/广告腔？
