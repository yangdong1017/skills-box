# dunya-sales

渡鸦科技社销转助手 Skill。

它用于把渡鸦产品矩阵、销转 SOP、异议处理库、产品手册、会员案例和销售理论封装成 Claude Code 可直接触发的销转参谋。

## 能做什么

- 分析客户画像和成交阶段
- 判断应该推荐哪个渡鸦产品
- 生成可直接复制粘贴的客户回复话术
- 处理价格、信任、适配、决策等常见异议
- 匹配相似行业/赛道案例
- 复盘销转对话，指出关键转折点和补救话术
- 判断同行探底、转介绍、老会员二转等特殊场景

## 典型触发

```text
/dunya-sales
/销转
帮我分析这个客户
这个客户怎么聊
该推什么产品
他说太贵了怎么回
客户说要考虑一下，怎么跟
有没有类似案例
帮我复盘这段对话
这个人是不是同行探底
```

## 使用方式

复制整个目录到 Claude Code 用户 skills 目录：

```text
C:\Users\<你的用户名>\.claude\skills\dunya-sales
```

或放到项目级 skills 目录中使用。

示例：

```text
帮我分析这个客户：他是做本地家政的，之前没做过小红书，预算不高，问我们588社群到底能不能帮他搞到客户。
```

输出会包含客户速判、推荐产品、可直接发送的话术、策略说明和下一步跟进建议。

## 使用原则

- 产品价格、边界、交付范围以 `references/01_核心层/产品矩阵与推荐决策系统.md` 为准
- 客户问具体产品细节时读取 `references/02_产品层/`
- 需要案例或数据时用 Grep 搜索 `references/03_素材层/`，不要整文件读入
- 深度心理分析或复杂成交策略时再读取 `references/04_理论层/`

## 发布说明

本 Skill 包含渡鸦内部产品、话术、案例、价格、转介绍与销转策略。适合团队内部或私有仓库管理。若公开发布，请自行确认授权和信息边界。
